## 一、核心方向

**本书核心不是修仙，是科技碾压。**

- 主角优势：现代军武记忆（国防大学研究生） + 功法提供的超长寿命（240年）
- 主线：利用现代知识研究超越时代的武器（火药、枪械、火炮等），以科技碾压这个世界
- 修炼是**工具**，不是主体。功法提供寿命和自保能力，但不占主线篇幅
- 修炼描写从简：感气→通脉→化罡 每个阶段只用1-2章概括性描写，重点放在突破时的**感受和意义**，不写修炼过程细节
- 科技线是**核心**：材料获取、工艺突破、武器试验、实战运用
- 权谋线：朝廷管制武者 → 主角以科技打破武力垄断 → 重塑世界格局
- 长生线：超长寿命让主角有时间从零开始研究一整套科技树

### 章节节奏
- 修炼章节：尽量压缩，1-2章过一个阶段
- 科技线：从第7章开始逐步介入，宋明作为知识伙伴
- 第21章下山游历时，科技线应有雏形（火药配方确认、简易火枪图纸）
- 第一卷结尾（第50章）：首次火力展示（火绳枪水平）

### 章节命名
- 章节名统一用**大于等于4个字**（允许4字及以上）
- 从第7章起执行（1-6章不改），后续新章一律≥4字
- 风格：平实直述，不文艺不卖弄，概括本章核心内容即可（如"初窥门径""童年·道心"）

### 男频爽文节奏公式（量化频率，❗对照自查）
| 层级 | 频率 | 本书落点 |
|------|------|----------|
| 小爽点 | 每 1–2 章 | 技术小突破 / 一个轻视者被噎 |
| 中爽点 | 每 3–5 章 | 以技破武碾压战 |
| 大高潮 | 每 8–12 章 | 幕内转折（狙击枪首杀宗师） |
| 卷终决战 | 每 15–20 章 | 卷四 571–650 决战听雨 + 北境钩沉 |

**四大放大器**：① 压抑前置+即时回报 ② 信息差爽点（主角/读者知、反派不知）③ 扮猪吃虎（肖凡文弱样）④ 数值化成长反馈（等级/装备/战绩可视）。
**v2 大纲自查**：每章须含 ≥1 小爽点（节拍「爽点标记」）；每 3–5 章须有一场以技破武的中爽点；节奏断档时回看卷首说明五幕骨架（§1.4 卷级核对表）。

## 二、科技设定

### 火药与武器研发约束
研发水平必须符合古代环境的最大可能，不能脱离实际：

- **核心设定：这个世界没有火药**。火药三要素原料都存在，但无人发现火药配方
- **硫磺**：本世界叫"石硫黄"，驱虫熏香，药铺有售，价格便宜
- **硝石**：本世界无人认识。主角需自行提取（土法炼硝：刮硝土→泡水→过滤→熬干结晶）
- **木炭**：日常劈柴即有
- **官府不管控**：石硫黄是普通商品，无任何管制
- **思路最稀缺**：世界不缺原料，缺的是"按比例混合塞进管子"这个思路

- **冶铁**：古代只有熟铁/生铁，枪管只能铸造后手工打磨，容易炸膛
- **火药**：只有黑火药。硝石手工提炼，纯度有限。配比靠经验，威力波动大
- **铸造**：手工模具，精度差。弹丸不圆→气密性差→射程精度大降
- **装填**：前装式，一分钟1-2发已是极限
- **击发**：火绳枪或燧发枪水平
- **供应**：没有工业化供应链，所有材料靠手工获取/制作

### 武器 vs 武者 威力设定
- **火绳枪/燧发枪（黑火药）**：对普通武者有效，对横练铜骨以上杀伤力大减，对宗师基本无效
- **高精度狙击枪（后装+膛线+无烟火药）**：对宗师构成致命威胁——超反应距离命中要害
- **研发时间线**：火绳枪→燧发枪（卷1）→膛线+定装弹药（卷2）→后装枪（卷3）→无烟火药+狙击枪（卷4杀手锏）
- **主角态度**：初期不确定火药能否伤到宗师，需亲眼见识后才能判断。后期逐步验证

> 修炼阶位每阶的「输入—过程—输出」明细见 `docs/governance/IDX_SETTING.md`「修炼体系」类子表，写越阶战前须核对，确保越阶有代价、非常态化（R4 锁）。

**第一卷结尾（15岁）能实现的合理上限：**
- 简易火绳枪（最原始的前装枪）
- 黑火药小批量自制
- 粗糙铅弹或铁砂
- 有效射程30-50步，杀伤距离约100步
- 可靠性差：容易哑火、炸膛，需要持续改进

**后续卷逐步升级：** 燧发枪→膛线→定装弹药→后装枪

### 资源稀缺度（低/中/高阶，严守 R2/R5）

| 层级 | 资源 | 掌控方 | 获取门槛 | 备注 |
|------|------|--------|----------|------|
| 低阶 | 铁料 / 黑火药原料（石硫黄·无管制、硝土、木炭）/ 熟铁 | 民间·工坊可控 | 低，手工可获 | 卷一即掌握 |
| 中阶 | 膛线工艺 / 定装弹药 / 后装枪 / 颗粒黑火药 | 肖凡班底 + 朝廷技术壁垒 | 中，需工艺积累 + 工坊体系 | 卷二/卷三解锁，朝廷欲垄断 |
| 高阶 | 无烟火药 / 狙击枪 / 玉佩灵物（灵脉·天材）/ 灵气火器核心 | 肖凡核心班底 + 内卫府双重把控；本世界无原生 | 高，配方·工艺绝密、原料稀缺 | 卷四解锁，国战级战略资源；私造按谋反论 |

- 低阶资源前期普及；中阶由技术壁垒控制；高阶须有「稀缺依据」——若某高阶资源后期变常见，必须补充分产地/技术改良/圈子升级等理由，否则削弱世界观稳定性。
- 高阶功法/资源由皇权与大宗门垄断，散修难越铜骨（普通人天花板参照 IDX_SETTING 玉佩七境表）。

### 第二卷·人才培养计划（重要）
- **核心目的**：一个人搞不了一整套工业体系，需要可信赖的班底
- **培养模式**：收养孤儿，长期教育为科研助手，数十年的渐进过程
- **教学内容**：现代数学、物理、化学基础
- **第一卷伏笔**：肖凡在第6-20章中观察到山下战乱/新政造成的孤儿现象，埋下"将来需要帮手"的念头

## 三、写作流程

### 大纲文件读取流程（重要）
- **按章节范围读取对应大纲文件**，不得一次性读取全部大纲
- 写第1-10章时，读取 `大纲\大纲_第01-10章.md`
- 写第11-20章时，读取 `大纲\大纲_第11-20章.md`
- 写第21-30章时，读取 `大纲\大纲_第21-30章.md`
- 以此类推，**每10章对应一个大纲文件**
- 如果对应章节的大纲文件不存在，先创建再读取
- 后续所有章节（包括第二卷及以后）都按此规律执行

### 写作前必读规则
- **每次写作前必须完整阅读本规则文件**，理解并严格遵守所有规则

### 大纲文件读取流程
- 写第1-10章 → 读取 `大纲/大纲_第01-10章.md`
- 写第11-20章 → 读取 `大纲/大纲_第11-20章.md`
- 写第21-30章 → 读取 `大纲/大纲_第21-30章.md`
- 写第31-40章 → 读取 `大纲/大纲_第31-40章.md`
- 写第41-50章 → 读取 `大纲/大纲_第41-50章.md`
- 以此类推，后续所有章节都按此规律读取对应大纲文件

### 章节正文写作·执行流程

**【第1步】读取大纲 + 时间线 + 反派设定**
→ 调用 Read，读取对应大纲文件（如 `大纲_第11-20章.md`）
→ 调用 Read，读取 `时间线与人物关系图.md`
→ 调用 Read，读取 `反派设定.md`（路径：`参考资料/反派设定.md`）
→ 确认：本章时间点、已发生事件、人物关系状态、反派动向

**【第2步】写入md草稿（直接写文件）**
→ 调用 Write，直接将章节草稿写入md文件
（如：`D:/个人/小说_道法百年/第一卷_道生/第0XX章_章名.md`）
→ 写入完成后告知用户"已写入，请审阅"

**【第3步】迭代修改**
→ 用户提出修改意见
→ 调用 Edit 或 Write 修改md文件
→ 重复直到用户确认

**【第4步】用户确认后 → 生成txt + 更新大纲 + 总结规则**
→ 用户说"确认/可以/生成txt"等明确用语后，才能进入此步
→ 调用 Write，写入同名txt文件（扩展名改.txt）
→ 统计字数（字符数/汉字数）并告知用户
→ 根据生成的txt正文内容，**反向更新对应大纲文件中本章的大纲内容**，确保大纲与正文一致
→ 调用 Edit 或 Write 更新 `时间线与人物关系图.md`
→ **总结本章所有修改意见，提取新写作规则**
→ **检查 `写作规则.md`，已有相同规则则忽略，无则追加写入**

**【第5步】迭代精修（可选）**
→ 用户对txt提出修改意见
→ 修改md → 同步txt → 告知用户

### 硬性禁止项（❗违反即流程错误）
- ❌ 用户未明确说"确认/可以/生成txt"前，**不可自行生成txt**
- ❌ 用户仅提出修改意见时，不可边改边生成txt——改完后再走"确认→生成txt"
- ❌ 扩写/修改已有章节时，同样须走完整流程

### 文件修改规则
- **重写大纲文件前必须备份**：用 Write 工具重写大纲文件（大纲_*.md）前，必须先创建备份文件（格式：`原文件名.bak.YYYYMMDD-HHMM`），防止数据丢失，方便回滚还原；章节正文写作（第*.md）按原流程执行，不要求备份
- **先md后txt**：先更新md文件 → 用户确认无误 → 再生成/同步txt文件
- **md修改后必须同步txt**，不能只改md不改txt
- **设定与大纲同步**：每次修改设定后，检查大纲文件并同步更新
- **生成txt时需统计字数**并告知用户
- **生成txt后更新 `时间线与人物关系图.md`**

### 扩写/修改章节同样遵守规则（❗注意加修）
- 对已有章节进行扩写（插入新段落、补充内容）时，**本规则文件所有条款同样适用**
- 必须逐条检查扩写内容：
  - ❌ 不与前文矛盾（前设后忘检查）
  - ❌ 逻辑因果清晰单一，无自相矛盾
  - ❌ 无刻意说明/替读者总结/盖章式结论
  - ❌ 无时间标记流水账（"第一天/第二天"）
  - ❌ 工具/物品获取路径合理
  - ❌ 转场自然不缺过渡
- 扩写完成后同样需要执行自查步骤，不能因"只是补充"而跳过

### txt文件格式
- 段落之间不保留空行，正常换行即可
- 每个段落独立一行，下一个段落另起一行
- 不将多个段落拼接在同一行

### 用户意见总结规则
- **每次用户提出修改意见后，必须总结并提取规则**，检查规则文件，不存在则新增

## 四、写作纪律（重要）

### 重复禁止
- 同一信息只说一次，说清楚了就过
- **重复高发场景（需特别警惕）**：
  - 对话中总结完→走后又内心推演同样的结论
  - 感慨类重复：同一个认知/感悟，用不同的话说了两三遍
  - 动作重复：同一动作先后写了两次（如"翻了翻"+"随手翻了几页"）
- 判断标准：删掉后文信息完整度不受影响，就是重复

### 信息获取逻辑
- 角色知道的信息在前文必须有获取路径
- **知识获取中介**：总数/概况信息需通过总纲、目录、前言等合理中介呈现
- 写之前先核对设定，确认信息获取路径的合理性

### 常见错误类型
- 角色突然知道某个事件/地名/人物（前文未铺设）
- 角色知道不该知道的信息（如7岁主角知道"这个世界连板甲都没有"）
- **前设后忘**：前文已铺垫的细节被后文推翻（如"装进一个竹筒"后面又说"分了几份"）
- 对话目的性太强：角色的措辞像替作者说明设定

### 写前检查（每次写新章节前）
1. 这条信息在前文哪章出现过？
2. 这个角色在当时有没有可能接触到这条信息？
3. 如果前文没有，角色的认知里就不能有。

### 写后自查（必须逐条完成，且必须在回复中贴出自查结果）

**❗强制执行**：每次提交草稿前，必须在回复中贴出逐条自查结果，格式如下。未贴出自查结果视为未自查，用户可直接打回。

① 逻辑核查：逐条对照前文设定，确保没有矛盾或推翻之处
② 重复核查：是否有同一信息反复出现（听说→心里→写出来三遍）
③ 前设后忘核查：物品状态/数量/位置前后一致
④ 年龄/时间核查：时间线与前章衔接无误，动作顺序符合逻辑
⑤ 作者视角核查：删除"替读者总结""否定对比""盖章式结论句"等解说性语句
⑥ 对话自然度核查：台词是否像说明书/列清单，是否符合人物身份
⑧ 时间标记核查：是否有"第一天/第二天"等流水账标记
⑧ 断句/拆句核查：一句话不超过50字，按动作/视线/情绪变化断句
⑨ 抽象词核查：禁止"很破""很冷"等抽象形容词，必须换具体细节
⑩ 因果链核查：每段开头是否是上一段结尾"触发"出来的，不凭空跳转

## 五、写作原则

### 衔接与转场
- 前后文自然衔接，不出现逻辑断层或跳跃
- **时间/空间转场必须有明确锚点**，不凭空跳转
  - 常见断点：对话→内心独白→站起来走（时序矛盾）
  - 常见断点：收功站直→盘坐在床上（缺"回屋"转场）
  - 忌用分割线"---"代替转场，分割线只用于大段切换
- 避免生硬的转折、总结或旁白式描述

### 场景化呈现
- 世界观、人物背景、设定信息，全部通过**具体场景和对话**呈现
- 不让角色做"背景板式简历介绍"，改由日常互动自然展现
- 参考《庆余年》开篇：用具体场景承载所有信息

### 情节丰富三要素（❗新规，对照自查）
情节单薄的表现是"跳跃式"——从A直接到B，缺乏过渡。丰富的故事要做到"连贯式"，每个核心情节都有铺垫、过程、余波。

**1. 前置铺垫**
- 关键情节发生前，用细节暗示、人物反应、环境烘托做铺垫
- 反例：主角突然打赢上届第一（无铺垫）
- 正例：主角先用功法A扭转劣势→对手反制→再用功法B→对手亮底牌→最后杀手锏定局。层层递进
- 好的铺垫还能制造"恍然大悟"的惊喜感，让读者回想前文时有"原来这里就在埋线"的感觉
- 避坑：铺垫信息要有用，不添加与主线无关的无用细节

**2. 增添过程**
- 不要直接描述"事情已完成"，要写出过程中角色的尝试、失败、挣扎、成长
- 反例："他去山洞炼硝，炼出来了。"
- 正例：火太大烧焦→骂自己一句→重来→慢火→成功（以细节增添层次感和代入感）
- 避坑：过程是为"让读者有代入感"服务的，不是为了写过程而写过程；注意控制篇幅，不要让细节拖慢节奏

**3. 延续余波**
- 每个事件都要有后续影响——情绪变化、关系改变、新矛盾、隐患
- 哪怕是小事，也要让它产生"涟漪"，让情节有联动
- 例：主角劈柴力气过大被师兄看见→以后劈柴得收着点（事件→应对），改变日常行为模式
- 一个情节结束后，问自己：这个事件之后，角色有什么变化？还有什么没解决？

### 不直白
- 用细节和动作暗示，不直接点破，让读者自己体会
- 网络热梗融入情节，不能刻意堆砌
- **日常对话要贴人设、合场景、不刻意**：活泼角色说话直白跳脱，沉稳角色说话温和内敛——切忌所有角色说一样的话
- **藏温情细节**：关心、牵挂、默契，不用角色直白说"我关心你"，而是通过简单的一句话、一个动作自然流露（如怼一下、揉头、撇嘴等小动作）
- **抽象词换具体细节（❗新规）**：禁止用"很破""很冷""很穷""很吓人"等抽象形容词，必须换成读者能感知的具体细节
  - 反例：`这是一间很破旧的屋子，生活条件很差。`
  - 正例：`墙皮一块块往下掉，窗框发黑，桌角缺了口，床边那盏灯亮一下灭一下，屋里还飘着一股潮湿的霉味。`

### 叙事视角统一
- 全文**第三人称有限视角**，以肖凡的感知为叙述镜头
- 禁止第一人称"我"的叙述方式
- 内心活动用第三人称表达，可省略"他想"直接呈现内心独白

### 禁止作者视角（❗重点）
- 从**读者和主角**的角度描述，不用作者的角度
- 不替读者解释"为什么"（如功法为什么用比喻）
- 不替读者总结"是什么"（如"这是基础功"）
- **结论由角色推导，不由作者喊出**：判断、决心、评价全部通过角色内心推演得出
- **禁止替读者总结**：读者已能自己得出的结论，不用再盖戳（如"他在心里排过去""难的是不让人发现"）
- **禁止否定对比式描述**：不用"不是A而是B"句式，直接写B
- **禁止盖章式结论句**：如"时机到了""确确实实存在"——前文已通过身体信号表达
- 判断标准：删掉后不影响叙事、只少了"解说"的，就是作者视角，应删除

### 描述自然不生硬
- 避免游戏式/系统提示式表述（"一行字浮现在眼前""解锁"等）
- **标点克制**：避免破折号（——），用逗号/句号/冒号替代
- **时间标记克制**：不用"第一天/第二天"流水账标记
  - 通过身体变化/季节更替/日常节奏自然带出（如"一晃几个月过去了""天渐渐冷了"）
  - 好的写法示例：用"有天晚上做完最后一式收功站直的时候"替代"练了一个月后"
- 禁止"清清楚楚""隐约""模糊"等替读者判断的形容词
- **断句（❗新规）**：一句话不超过50字，按动作变化/视线变化/情绪变化断句，给读者缓冲
  - 反例：`他推开门之后看到客厅里一片狼藉地上到处都是碎玻璃而母亲坐在沙发边低着头一动不动他心里一下就慌了。`
  - 正例：`他推开门，客厅一片狼藉，地上全是碎玻璃。母亲坐在沙发边，低着头，一动不动。那一瞬间，他心里猛地一慌。`
- **拆句（❗新规）**：一句话信息太多时，先写主干动作，再补心理/原因/状态
  - 反例：`她在看见走廊尽头那扇本该锁着的门竟然半掩着之后，呼吸一下就停住了，站在原地不敢继续往前走，脑子里飞快闪过昨晚听到的那些传言，越想越觉得后背发凉，连手心都渗出了汗。`
  - 正例：`她猛地停下脚步，盯住了走廊尽头那扇门。门竟然是半掩着的。她一时没敢再往前走。昨晚听来的那些传言，忽然全涌了上来。她越想越冷，连手心都慢慢渗出了汗。`

### 因果链（❗新规，解决衔接不自然）
- **每段开头必须是上一段结尾"触发"出来的**，不是作者强行切换场景
- 用"触发→反应"链条代替孤立的场景描写
- **操作方法**：写完一段后，检查下一段开头是否有"因为上一段，所以这一段"的逻辑链
- **反例**：`[结尾] 胖子大步走了。` → `[下一段] 街对面有个包子铺……`（无因果，跳接）
- **正例**：`[结尾] 胖子睨了他一眼，骂了句"贱骨头"，大步走了。` → `[下一段] 肖凡的目光追着那背影看了一会儿，转到街对面……`（目光触发→转场）
- 无法建立因果链的两段之间，必须加一个"过渡动作"（如：肖凡收回目光/抿了抿嘴/走了一段……）

### 环境/氛围描写：增加画面感与情绪
- 环境描写不是"多余的话"，而是故事的调味剂——烘托情绪、渲染氛围、让场景有画面感
- 好的环境描写能让读者"身临其境"：不是告诉读者"女主很伤心"，而是写"夕阳把河水染成橘红色，芦苇在风中沙沙作响"
- **必须贴合人物情绪和情节**：角色紧张时，环境可以阴暗压抑；角色放松时，环境可以温暖明亮。不能脱离故事空谈风景
- **篇幅控制**：超过100字的环境/情绪描写要思考是不是水文了。环境描写是调味剂，不是主菜
- 不同场景侧重不同：山野场景侧重自然风貌、市井场景侧重烟火气、道观场景侧重宁静古朴

### 功法/知识类描写
- 通过主角的试炼体验呈现，不通过叙述直接总结
- 招式列表、知识碎片自然融入叙事流，不另起段落刻意标注

### 修炼/战斗描写：写结果和感受，不写过程
- 不逐式逐招展开。用一个场景、一个感受、一个结果代替流水账
- 战斗写气势、心理、后果，不写每一招每一式
- 修炼的核心不是"怎么练的"，而是"为什么能成"或"为什么卡住了"
- 参考《庆余年》：范闲修炼全程用"午睡即修炼+不怕所以通关"两个点说清

### 战斗分层标注（❗v2 大纲须标，对照自查）
每场战斗按五层推进，避免"一招定胜负"式平铺，让爽感有释放点：
- **试探**：亮底牌、制造信息差（火器初见，对手以为"铁管子"是儿戏，呼应 R4 压制宗师无效的反差铺垫）
- **交手**：拉扯，让危机真实（铜骨武者逼近，主角且战且退，靠地形/工事周旋）
- **转折**：一张牌翻盘（灵气火器附刃 FB-049 突然破防）
- **爆发**：爽感释放（狙击枪一发 FB-065 贯穿宗师——认知被颠覆的瞬间）
- **收尾**：余波+代价（暴露技术、引来更高层注意，呼应 §五「延续余波」）
节奏公式：**信息差 → 压制 → 反转 → 碾压/惨胜**。
**越阶战频率严控（R4/R5）**：火器对宗师基本无效，狙击枪 FB-065 是卷四唯一例外，破宗师后不可常态化，否则等级压制崩盘。多线编排：战场/暗线/朝堂同步，交汇处放大爽感。
**v2 大纲写法**：战斗章节拍「阶段标签」须从「试探/交手/转折/爆发/收尾」枚举中选其一（见 OUTLINE_FORMAT §六 B4）。

### 节奏张弛：故事要有紧有松（❗新规）
一直快→读者疲惫；一直慢→读者乏味。丰富的故事节奏一定是张弛有度的。

**常用手法：**
- **高潮细化**：大场面（冲突/战斗/重大发现）把细节写足，有"盛宴感"
- **穿插日常**：高潮之间穿插轻量日常（干活、做饭、闲聊），给读者喘息空间
- **藏重线于闲笔**：日常中藏伏笔，让读者放松时也不觉得"这章没推进"
- **适度反转**：适当加入小反转（如主角以为某事顺利，实际出了意外），增强期待感，但不宜频繁导致疲劳
- **短平快日常**：温馨对话以短句为主，加语气词和小动作，篇幅不宜过长

### 动作与逻辑一致（❗重点）
- 动作顺序符合逻辑（先披外袍再推门）
- **工具/物品获取路径**：角色使用的工具必须是当时能获取的（如山洞炼硝用木条，不用小刀）
- **角色身份限制**：动作和知道的不超出角色年龄/阅历
- **因果关系明确**：一个动作的原因和结果清晰，不能同时为两个冲突的目的
- **前设后忘禁止**：前文写过的物品状态、数量，后文不得矛盾
- **用身体语言代替情绪词**：不说"他很生气""她很伤心"，用动作和微表情呈现——指尖攥紧、眼底泛红、声音发抖、呼吸急促。身体语言比情绪词更有画面感

### 视角转换自然
- 不写"这是……""稳重/精明"等直接定性的话
- 不用"不久之后""日子一天天过去"过渡带情节
- 不用先给结论后写过程的写法
- 人物性格融进具体场景和对话里

### 权谋描写：信息不对等驱动
- 权谋冲突来自**信息差**，不来自善恶标签
- **层层揭秘**：每次只多给一个信息点，不给全套答案
- **潜台词**：关键信息点到为止，留给读者想象
- **闲笔藏重线**：重要信息用最平淡的语气说出
- 目前可埋长线：灭门案幕后势力、玉佩来历、无为子隐藏身份、朝廷管控武者的真实动机

### 布局与埋坑
- **先果后因**：先展示结果，原因逐步揭晓
- **沉默即伏笔**：人物长期不出场本身就是伏笔
- 线索跨章分布式放置，收线时让读者拼出完整画面
- **人物过往分步释放**：角色的过往经历（童年、创伤、执念）不要一次性全盘托出，分多次穿插在情节中，每次只放出一小片——既解释人物动机，又保留悬念让读者持续探究
- 避坑：不要添加对剧情推动或人物塑造没有意义的回忆片段，也不要释放得太过频繁

### 语言风格
- 半白半文，古雅不晦涩，不用之乎者也堆砌
- 对话偏古风，叙述偏现代流畅
- 氛围感：肃杀、沉稳、有历史厚重感
- 用词符合时代背景（永熙年间为稳定时期，不用"乱世"等词）
- **内心独白用现代人思维**：肖凡内心想法用现代语气（"武功再高也怕菜刀""活不过三集"），对外说话偏古风

### 风格融合（外部作品提取 · 持续累积）
- **调用文件**：`参考资料/风格融合手册.md`（唯一累积源）。每用户提供一部外部小说，AI 提取其风格，把**可落地笔法**合并进该手册「一、已沉淀笔法」，写章前 consult。
- **当前已沉淀笔法（摘要，详参手册）**：① 限知+史笔双层（章首/转场「永熙X年，大晟……」定调，年号≤37）；② 慢热日常密度（清虚观日常铺足）；③ 碎片记忆蒙太奇（受创/冲突用碎片+俯瞰，不泄科技）；④ 专业视角转译（肖凡用「前世所受训练/阅历」解剖人际，不点现代）；⑤ 神态动作标签密（对话必带神态/动作）；⑥ 冷面打趣+潜台词（肖凡偶带幽默，藏机锋）；⑦ 冷峻+温情闪现（借细节不抒情）；⑧ `……`留白悬念。
- **应用铁律**：所有外来笔法**不得突破**红线 R1–R3/R14/R17/R19、年号≤37（C3）、空间锚点含清虚观/盛京（C7a）、FB-002 穿越保密。手册「二、本作硬约束」为兜底。

## 七、写作禁忌

### 格式混乱
- ❌ 分章异常、标点符号错乱、排版格式错乱
- ❌ 短篇童话、公文报告等非长网文体裁叙事方式

### 结构失常
- ❌ 梗概式写文——把章节当成大纲复述，没有场景和细节
- ❌ 开篇结尾句式重复堆砌，模板化拼接（如每章开头"第二天……"、结尾"他心想……"）

### 空洞水文
- ❌ 机械罗列时间、动作、对话的流水账，无实质情节推进
- ❌ 行文空泛，节奏拖沓，长篇幅无有效信息
- ❌ 堆砌艰涩抽象的高级概念，牺牲可读性

### 避坑·丰富陷阱（对照自查）
- ❌ **不写无关内容**：所有拓展的内容（回忆、配角、环境、细节）都要贴合主线或人物，不能凭空加一段和故事无关的情节
- ❌ **不重复冗余内容**：同一类细节、同一类情绪，不要反复写（如每次伤心都写"她哭了"→改用不同细节表达：眼底泛红、转身避开视线、指尖攥着衣角）
- ❌ **不脱离人物逻辑**：角色的言行要符合其性格和动机。冷漠寡言的角色突然变得唠叨热情，必须有合理原因
- ❌ **不破坏节奏**：不要在紧张冲突中强行插入日常对话或大段环境描写
- ❌ **日常对话不宜过长**：温馨小对话短平快，简单且有趣，篇幅过长即成水文
- ❌ **表述冗余（❗新规）**：一句话能说清的，不要用三句话绕
  - 句子越绕，节奏越慢，情绪越散
  - 反例：`她在听完这句话之后，内心当中立刻就产生了一种非常明显的慌乱情绪，整个人一下子不知道该怎么办才好。`
  - 正例：`她一听这话，心口猛地一乱，站在原地都忘了接下一句。`

## 八、四大支线风格

### 1）武侠线：金庸正统风
- 内力、招式、门派、江湖规矩
- 打斗重意境、简洁、一击制胜
- 不玄幻、不爆种、不修仙

### 2）权谋线：《大明王朝1566》质感
- 朝堂博弈、党争、皇权制衡
- 台词锋利，暗流涌动
- 重制度、重人心、重利益

### 3）科技线：硬核工业党文风
- 不讲玄学，只讲材料、工艺、原理
- 火药、枪械、蒸汽机：写得真实可感
- 用"细节"体现碾压，不用喊口号

### 4）长生线：时间沧桑感
- 每次苏醒：物是人非
- 突出"我自岿然不动"的孤独与强大
- 遗憾情感线 + 三百年孤独，风格：冷、稳、沉、远
- 少甜宠，多"身不由己"与"大道无情"

## 九、主角设定

### 穿越方式
- 被遗弃时已是穿越者，拥有现代记忆（国防大学轻武器研究生）
- 婴儿期：有现代记忆，能清晰思考，但身体无法表达
- 幼儿期：有记忆但需隐藏
- 觉醒感：不需要觉醒，本身就有记忆

### 主角口吻
- 极度冷静、心思极深、话少事多
- 性格：冷眼观世、算无遗策、情绪克制，只在关键节点流露
- 内心戏偏：战略、制度、科技、势力控制
- **内心独白用现代人思维**，不文绉绉
- **网络热梗可入内心戏，但必须自然**：不让角色专门停下来"说梗"，而是在思考过程中自然带出

### §九-附 主角例外机制（❗绑定 R4/R5，防止能力失控）

| 项目 | 内容 |
|------|------|
| 来源 | 穿越者 + 玉佩双核心：现代军武知识（国防大学轻武器研究生）＋ 玉佩功法（无暗伤长生 + 灵气）。本世界独有，不可复制 |
| 优势 | 降维打击：以工业党体系造出本世界无原生火药的火器；玉佩功法提供无暗伤长生，支撑从零研发整套科技树 |

### 配角写作原则
- **配角要有自己的欲望**：每个人物都有自己的目标、烦恼和利益考量，不单纯为主角服务。哪怕欲望很小、很卑微（如想学更多知识、想赚钱养家），也能让配角显得真实
- **每个配角有一个核心标签**：一个核心特质+1-2个辅助细节，就能让读者记住（如林谋=机敏善谋+跑山下经商，宋明=书痴+温润博学）
- **配角出彩不是抢戏份，是有辨识度**：有记忆点但不盖过主角，大场面以主角为中心
- **不宜重复雷同配角**：不要写多个性格和职能雷同的配角

### 冲突关系网（❗非工具人强化）
每个重要角色都有独立目标，且与主角之外的角色也存在**多边拉扯**（结盟/制衡/反目/误会），不靠主角单独成立。四问建模：① 核心诉求 ② 底层动因 ③ 与主角冲突落点 ④ 与其他角色的多边关系。
- 本书示例：苏墨凝（查父案）vs 青溪派（旧怨暗线）→ 主角夹在中间，关系复杂化；老柯（手艺被认可）vs 内卫府军械官（理念冲突）→ 班底与朝廷的张力。
- 反派同理：顾衡灭门非为针对主角，而是为灭口自保（见 反派设定 §2.5）；裴琰执行萧沛遗策有独立出路考量（非"哪里有主角就去对付哪里"）。
- 写前自检：该角色的冲突是否能在"主角不参与"时依然成立？若只能靠主角推动，则仍属工具人，须补独立动机。

## 十二、对白写作（❗新规，来自写作资料）

### 人物辨识度
- 一个人的身份、年龄、脾气、文化水平、说话习惯，直接影响措辞和语气
- 优秀对白：人物一张口，读者就能从说话方式里感受到"这个人是谁"
- **年龄越大、身份越稳的人，通常更少说废话，更少把情绪摊开说**
- **亲近关系不代表说话一定温柔**，很多亲情恰恰通过责备、打断、嘴硬表现出来
- 反例：`父亲说："我希望你以后能够更加重视自己的身体健康，毕竟昨天晚上你工作到太晚了，这种行为是不正确的。"`
- 正例：`父亲皱着眉，把桌上的豆浆往他面前一推："一宿没睡？"`

### 互动感（❗重点）
- 有互动感的对白，重点不在于说了多少，而在于人物有没有对彼此做出反应
- 这个反应可以是：接话、打断、反问、否认、顺势嘲讽、故意岔开，也可以是动作和表情上的即时回应
- **互动感的核心是"回应"，不是"轮流发言"**
- 对白里可以加入：打断、追问、抢话、吐槽，让人物活起来
- 反例（轮流发言，无互动）：
  ```
  林舟说："你今天怎么来得这么晚？"
  周凯说："因为路上堵车了。"
  林舟说："原来如此，那我们现在进去吧。"
  周凯说："好的，我们进去。"
  ```
- 正例（有来有回，有活气）：
  ```
  林舟看了眼时间："你再晚五分钟，我都准备给你立碑了。"
  周凯把车钥匙一甩："少来，堵了一路，我能活着到这儿已经算命大。"
  林舟挑眉："你上次迟到也这么说。"
  周凯面不改色："说明这理由稳定，值得长期使用。"
  ```

### 推动剧情（❗重点）
- 对白不是摆设，不能只负责"热闹"，还应该承担任务
- 这个任务可以是：抛出新信息、制造误会、加深冲突、做出决定、改变关系，也可以是推动人物进入下一个行动
- **每段关键对白，最好至少承担一种功能**：给信息、造冲突、做决定、改关系、推行动
- **对白结束时，局面最好发生一点变化**，哪怕只是人物态度变了，也算推进
- 反例（有敌对气氛，但没有新信息，没有局势变化，属于原地打转）：
  ```
  陆沉冷笑："你果然还是来了。"
  裴烬看着他："我当然会来。"
  陆沉说："来了也没用，你赢不了我。"
  裴烬神色冰冷："那可未必。"
  ```
- 正例（抛出了新信息，制造了时间压力下的强迫选择，直接把双方从对峙推入生死交锋）：
  ```
  陆沉立在祭坛高处，垂眼看他："你来晚了，镇魂印已经开了两层。"
  裴烬脚步一顿，目光猛地落向他身后的血阵："你拿百姓祭阵？"
  陆沉笑了一声："不然你以为，城南那一百三十七口人，是怎么一夜失踪的？"
  裴烬手指骤然收紧，声音沉了下去："他们果然在你手里。"
  "现在才猜到？"陆沉抬手一挥，石门轰然落下，"可惜，退路没了。要么杀了我，毁阵；要么再拖半刻钟，等着给他们收尸。"
  裴烬盯着他，缓缓拔剑："那你今天就别想活着走出去。"
  ```

### 潜台词（❗重点）
- 人物可能嘴上在抱怨，实际是在关心；嘴上说无所谓，实际是在试探
- 潜台词能让对白更像真实的人在说话，也能让读者参与理解，从而增强阅读黏性
- **越重要的情绪，越不要第一句就说破**
- 潜台词常常藏在：反话、停顿、答非所问、刻意平静里
- 反例（把情绪说明书式地讲完了，读者没有任何体会空间）：
  ```
  女生说："我现在很生气，因为你昨天没有回复我的消息，而且你和别的女生一起吃饭，我吃醋了。"
  男生说："对不起，我知道你吃醋了，我以后不会这样了。"
  ```
- 正例（女生没有直接说"我吃醋了"，但情绪非常清楚。对白有了留白，读者会主动读出弦外之音）：
  ```
  她低头搅着杯子里的冰块，语气很淡："昨天挺忙啊。"
  "手机没电了，后来应酬也没顾上看。"
  "哦。"她点点头，"我还以为，你跟那位一起吃饭，开心得忘了我呢。"
  他听出不对，往前坐了点："你看见了？"
  她终于抬眼看他："看见倒没什么，主要是某些人，连解释都省了。"
  他沉默两秒，低声说："是我不对。"
  ```

## 十六、大纲撰写流程

### 核心原则
- **大纲草稿必须先在对话中输出，经用户审阅确认后，才能写入文件**
- 任何情况下，只要用户只提出修改意见、没有说"确认写入"或"写入文件"，都不得写文件

### 严格执行步骤
**第一步：输出草稿** → 在对话中输出大纲草稿，供用户审阅

**第二步：用户反馈** → 用户提出修改意见，我迭代修改草稿

**第三步：用户确认** → 用户明确说"确认写入"/"写入文件"/"可以写入"等明确用语后，才能进入下一步

**第四步：写入文件** → 将确认后的大纲写入对应大纲文件

**第五步：写入后汇报** → 告知用户文件已更新，并附上写入的内容摘要

### 硬性禁止项
- ❌ 用户仅提出修改意见时，不得写入文件
- ❌ 用户说"继续"/"好的"/"知道了"等模糊用语时，不得视为写入许可
- ❌ 不得因"上一轮用户提供了内容"就默认为已确认
- ❌ 不得将用户的修改意见当成写入指令执行

### 适用范围
- 所有新章节大纲
- 所有已有大纲的修改
- 所有大纲文件的任何写入操作

### 文件管理
- 大纲文件按章节范围存放：`大纲_第XX-XX章.md`
- 写入前必须先读取对应文件，确认当前内容
- 写入时必须用精确匹配，不得误删或覆盖其他章节内容

### 大纲写作标准（❗重点）

大纲是**剧情骨架**，不是章节标题或主题概括。每一份大纲都必须达到以下标准：

**6项核心内容（硬性标准）：**
任何一章的大纲，只要包含这6点，就是专业级细纲：

1. **本章核心任务**（最重要）：这一章必须完成的剧情目的，只能有1个主任务，最多2个辅助任务。作用：杜绝水文，每一章都有价值。
2. **本章冲突/矛盾**：一章必须有冲突（人际冲突、危机冲突、选择冲突、压力冲突）。无冲突=纯水章，细纲必须提前写清楚：谁为难主角、什么压力、什么困境。
3. **关键情节流程**（分步剧情）：把一章内容拆成3–5步小剧情，按顺序写：开场→发展→交锋→收尾。写手直接照着分段写正文，节奏完美。
4. **人物情绪/心态变化**：本章主角心态（隐忍、愤怒、警惕、得意、凝重、绝望等）。保证人物不OOC，情绪连贯。
5. **本章收获/代价**：主角每一章必须有结果——收获（情报、人脉、资源、实力、名声）或代价（暴露风险、结仇、消耗、埋下隐患）。
6. **伏笔/下章钩子**（留人关键）：章节结尾必须留悬念、危机、未知、新麻烦、新机遇。这是读者追读的核心。

**格式要求：**
- 每个要点都是一个**完整事件**，有明确的发生过程和结果
- 不用笼统的描述，要写清楚"先发生什么，再发生什么，导致什么结果"
- 示例（❌反例）：`开业当天生意很好，客人赞不绝口`
- 示例（✅正例）：
  ```
  - **本章核心任务**：天上人间开业，通过服务和新菜碾压醉仙楼，确立京城新贵地位
  - **本章冲突**：钱万山安插眼线捣乱、国师府突然送礼引发猜测、醉仙楼暗中观察
  - **关键情节流程**：
    1. 造势完成，遮布揭开，客人涌入
    2. 国师府送礼，全场震惊，肖凡身份成谜
    3. 服务和新菜引发轰动，食客赞不绝口
    4. 醉仙楼客流锐减，钱万山摔杯怒吼
  - **人物情绪变化**：肖凡从平静观察→确认成功→暗自警惕；钱万山从观望→震惊→暴怒
  - **本章收获/代价**：收获（天上人间一炮而红、国师府背书、京城名声）；代价（与钱万山矛盾激化、被东宫势力注意）
  - **伏笔/下章钩子**：钱万山暗中联络东宫，准备对肖凡下狠手
  ```

**自查清单：**
写入大纲后，逐条检查：
1. 是否明确了本章核心任务（1个主任务，最多2个辅助任务）？
2. 是否写了本章冲突/矛盾（谁为难主角、什么压力、什么困境）？
3. 是否拆分了关键情节流程（3-5步小剧情，开场→发展→交锋→收尾）？
4. 是否写了人物情绪/心态变化？
5. 是否写了本章收获/代价？
6. 是否留了伏笔/下章钩子？
7. 写正文的人拿到这份大纲，能否清楚知道每一段该写什么？

## 十七、第9章提炼规则（新增）

### 人物描写规则
- **写人物必须先读最新设定文件**，不能自行脑补人物背景（如二皇子"军功在身"为AI自行添加，设定中并无此内容）
- **次要年幼人物一句话带过即可**，不必展开（如四皇子萧俊，6岁，仅需提及即可）

### 信件与隐私
- **他人信件/引荐信，主角不能私拆**，相关内容不向主角透露，由交信人当面说明情况即可

### 描写禁忌
- **禁止大段内心独白直接说明动机**（如"他来这里，不是为了投奔谁，也不是为了..."），动机通过行动和自然流露呈现
- **禁止作者替读者解释的解说句**（如"说书先生没存心告诉他这些，但..."），删掉后不影响叙事的就是多余解说
- **避免总结性描述**，改用具体环境/动作描写（如"他身后是一条走了五个月的路"改为环境描写）
- **同一抒情点不能重复**（如"京城，我来了"在一章内出现两次）

### 信息重复禁止
- **同一信息不在"听说→心里→写出来"重复三遍**，听到后心里记下即可，不必再写一遍
- **同一场景内相同信息只能写一次**（如前面已算过盘缠，回房后不再重复算）
- **避免重复描述**（如通州镇大小前后重复、道观对比在下山多个月后已不合适）

### 说书内容规范
- **说书人讲趣事，不讲城市布局/城防图**，信息从闲话中自然带出，让主角自己拼凑

### 对话自然度
- **对话要有语境，不能写不通的话**，说书场景中的附和话要符合听众身份和语境（如"不急不急"无语境则删改）

### 逻辑一致性
- **改一处须同步检查同场景相关描述**（如"住两晚"与"明天进京"矛盾，须统一）
- **生成txt须清除md专用标记**（如"（本章完）"等markdown章节结束标记不能进入txt）

### 写作流程补充
- **写作流程第1步必须读反派设定文件**（`参考资料/反派设定.md`），确保人物行为与设定一致
