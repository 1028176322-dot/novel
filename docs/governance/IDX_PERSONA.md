# 角色性格与行为逻辑总表 IDX_PERSONA

> 版本戳：2026-07-22 覆盖 1–1000 章（单文件全量，不按卷拆分）
> **权威数据源**：角色"性格—行为"一致性的唯一判据。与 `IDX_CHARACTER.md`（出场/状态/禁复活）互补——CHARACTER 管"在不在、死没死"，PERSONA 管"像不像、该不该这么干"。
> **接入点**：① Stage1 必读第 11 项（撰写前必须加载）；② Stage2 动笔前"人设行为边界预检"逐角色取本表输出《人设行为备忘》；③ Stage3 逻辑矩阵 L4「角色性格与行为逻辑一致性」逐角色对照本表判定 OOC。
> **核心原则**：每个角色都是独立人格，言行须由其性格 + 处境驱动，禁止"作者需要他怎样就怎样"（OOC）。主角肖凡是**现代灵魂**，言行须带现代思维痕迹（仅限内心独白与反差爽点，绝不破古代氛围）。

## §1 主角·肖凡（现代灵魂）行为守则

| 维度 | 规范 |
|------|------|
| **身份锚** | 穿越者灵魂（FB-002 绝对保密）· 清虚观弟子 · 科技+修炼双线 · 藏拙布局者（R17） |
| **核心性格标签** | 冷静理智 · 务实 · 外冷内明 · 藏拙不抢戏 · 现代思维内核 |
| **现代灵魂·说话风格** | 对外半文半白**适配古代场合**（不突兀）；台词简洁、留余地、偶有反问；现代考据/逻辑只在**内心独白**展开；不直白掉书袋，引经据典是"观中习得"的借用而非真通古文；幽默偏**冷峻自嘲式**（现代式），非古风俏皮 |
| **现代灵魂·做事风格** | 效率与结果导向（现代管理思维）；**系统组织**（班底分工如现代公司：老柯管造/沈遇管账/苏墨凝暗线，各司其职）；重视情报与验证（不信空口、重证据链）；借势不盲从权威（利用皇权/国师但保持独立）；实用主义（不纠结虚名虚礼）；强风险意识与预案（布局者 R17）；**用现代认知解古代难题**（科技+修炼融合、工程、情报网）——这是"现代 vs 古代"反差爽点的根本来源 |
| **现代灵魂·经营理念** | 现代电商/管理思维驱动，**刻意不写古风老字号套路**：**效率分工**（班底各司其职、工坊流水线化，以组织力碾压手作单干）；**技术壁垒护城河**（独门工艺/配方即门槛，竞者难以复制，如酒行压价、火器量产）；**品牌与口碑**（货真价实、人无我有，"好物"即硬货标准，靠复购与口碑滚雪球而非一锤子买卖）；**供应链与分销**（拢货一站式、跨地行商网摊薄成本、以量换价）；**制度降维**（以商籍/工籍/契约分红等规则与班底架构碾压对手，而非靠人情江湖或单打独斗）。实战中以现代经营逻辑降维古代商号——"好物商城"仅为现代思维的表，内核是上述五条 |
| **现代痕迹边界（红线）** | 内心可自由用现代概念（枪/机械/管理/化学直觉）；**叙事旁白与台词零现代实物名词、零网络梗**（红线 R19/文风基线）；绝不向任何角色泄穿越（FB-002 绝对保密）。**R19 例外**：肖凡商号专有命名（好物商城等）作为叙事内字号，属 R19 例外——仅限商号牌匾题字、角色对话中"好物商城"等店名提及，不扩散为现代实物名词、不夹杂网络梗解说；后续专项店/分号各取不同现代商号名，同样仅作叙事内命名使用 |
| **性格成长弧** | S0 弃婴（现代意识+婴儿躯，内心吐槽世界） → S1 童年（内敛观察、藏锋） → S2 游历（务实谋断、始布局） → S3 中年（沉稳布局者，科技修炼双线碾压） → S4 老年返童（通透，现代内核不灭） |
| **不崩底线** | 藏拙不抢戏（R17）；不滥威；反派不降智衬托；因果链完整；不写角色认知域外信息（见 `IDX_COGNITION.md`） |
| **易踩 OOC 坑（教训）** | ① 让肖凡台词突然蹦现代词/网络梗（破氛围）；② 婴儿期写"懂世界全貌/火药缺失"等穿越瞬间不可得信息；③ 为爽而让肖凡当众炫技破藏拙；④ 把"现代思维"写成旁白解说而非内心活动 |

## §2 主要角色性格档案（多元化 · 独特详描）

### §2.1 清虚观系（师门）

**无为子（观主·师父）**
- 性格标签：**内敛沉稳 · 慈严 · 护短** · 不善权谋

- 雷区：不可写其参与朝堂权谋或识破肖凡底牌；不可写其急躁冒进
- 与主角：慈父式师长，肖凡敬而不依赖

**陈守（大师兄）**
- 性格标签：**沉稳寡言 · 长兄如父 · 责任感强**
- 说话风格：惜字如金，多用行动代言语；偶尔一两句沉甸甸的叮嘱
- 行为/决策：默默扛事、护弟妹；不争不抢
- 雷区：不可写其话痨或轻浮

**林谋（二师兄）**
- 性格标签：**机灵跳脱 · 少年气 · 带点狡黠的热心**
- 说话风格：快人快语、爱打趣；带肖凡下山时的"小大人"腔
- 行为/决策：活络、爱跑腿传话；遇险先喊后想（非真慌，是反应快）
- 雷区：不可写其老成持重（那是陈守）；可作为"幽默/跳脱"维度担当，但不过度插科打诨抢主线

**周猛（三师兄）**

**宋明（四师兄·书痴）**

- 说话风格：引经据典、爱纠错；论典时眼睛发亮
- 行为/决策：钻书堆、以典籍为准；肖凡的"知识伙伴"

## §3 反派性格通则（不降智 · 有自身逻辑）

1. **反派不是蠢，是立场/信息差导致失败**；其行动在自身认知域内合理。
2. **有独立动机**：夺嫡、自保、部族存亡、利益——绝不"为坏而坏"。
3. **不送爽点**：反派决策须让主角"赢得有难度"，衬托不出主角强的降智写法即 OOC（违反读者视角"爽点释放"项）。
4. **信息不全**：反派不知肖凡全部底牌（FB-002 保密）、不知未回收伏笔答案。
5. **退场有尊严**：被技服/罪定/潜踪，非儿戏横死；禁复活角色零复活。

## §4 性格—行为一致性写作核查（逐角色自检）

撰写每章时，对**每个出场角色**逐条回答如下问题（任一项"否"即 OOC → FAIL 回改）：

- [ ] 该角色言行是否命中其 §2 性格标签（主标签至少成立）？
- [ ] 说话风格是否匹配其"声口特征"（肖凡现代思维仅内心；道门半文半白；市井俗白；边军军旅腔）？
- [ ] 行为/决策是否由其性格 + 当前处境驱动（非作者为了方便强行安排）？
- [ ] 是否触及该角色"雷区/禁忌"（如萧俊知毒、无为子参与权谋、反派降智）？
- [ ] 主角肖凡：现代灵魂痕迹是否仅限内心/反差爽点，台词旁白零现代实物名词（R19）？
- [ ] 主角肖凡：是否维持藏拙（R17）、未泄穿越（FB-002）、言行符合其年龄阶段性格弧（§1 成长弧）？
- [ ] 新出场角色：是否带了"独特性格标签"，而非无差别工具人？

## §6 写法约束速查
