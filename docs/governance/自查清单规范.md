# 自查清单规范（唯一权威源）

> 本文件是《道士也玩火力覆盖》写作自查的**唯一权威清单**，由 novel-chapter-write 四阶段流程阶段三强制引用。
> 与《AI写作自动化流程.md》Stage3「13 维 + 读者视角 12 项 + L1–L9 逻辑矩阵」为同一总检口径的**压缩可执行版**；写作规则.md §四 不再重复正文，仅指向本文件。
> 版本戳：2026-07-22（随 logic_check.py C1–C22、流程文档 13 维同步）；2026-07-24 升 C18–C22 行（补 FB码泄漏/七境回归/引荐信冲突/段落重复/句级重复，均取自 `logic_check.py` 权威定义）。

---

## 一、写前检查（每次写新章节前必过 3 问）

1. 这条信息在前文哪章出现过？
2. 这个角色在当时有没有可能接触到这条信息？
3. 如果前文没有，角色的认知里就不能有。

## 三、写后自查 10 条（提交草稿前必须逐条贴结果）

```
自查结果：
① 逻辑核查：通过 / 未通过（说明）
② 重复核查：通过 / 未通过（说明）
③ 前设后忘核查：通过 / 未通过（说明）
④ 年龄/时间核查：通过 / 未通过（说明）
⑤ 作者视角核查：通过 / 未通过（说明）
⑥ 对话自然度核查：通过 / 未通过（说明）
⑦ 时间标记核查：通过 / 未通过（说明）
⑧ 断句/拆句核查：通过 / 未通过（说明）
⑨ 抽象词核查：通过 / 未通过（说明）
⑩ 因果链核查：通过 / 未通过（说明）
```

判定细则：
- ① 逻辑核查：逐条对照前文设定，确保没有矛盾或推翻之处
- ② 重复核查：是否有同一信息反复出现（听说→心里→写出来三遍）
- ③ 前设后忘核查：物品状态/数量/位置前后一致
- ④ 年龄/时间核查：时间线与前章衔接无误，动作顺序符合逻辑
- ⑤ 作者视角核查：删除"替读者总结""否定对比""盖章式结论句"等解说性语句
- ⑥ 对话自然度核查：台词是否像说明书/列清单，是否符合人物身份
- ⑦ 时间标记核查：是否有"第一天/第二天"等流水账标记
- ⑧ 断句/拆句核查：一句话不超过50字，按动作/视线/情绪变化断句
- ⑨ 抽象词核查：禁止"很破""很冷"等抽象形容词，必须换具体细节
- ⑩ 因果链核查：每段开头是否是上一段结尾"触发"出来的，不凭空跳转

## 四、阶段三总检口径（压缩，完整定义见《AI写作自动化流程.md》）

- **13 维规则核验**：格式/设定/纪年/空间/人物/伏笔/跨章/手法适配/跨章防雷同/…（详见流程文档）
- **读者视角审查 12 项**：应见 / 禁见 双列（含正文纯净度、现代词仅限内心独白等）
- **逻辑一致性防御矩阵 L1–L9**：L1 角色认知边界（全员·分层）/ L2 时间一致 / L3 物品物理 / L4 角色性格与行为逻辑 / L5 设定世界观 / L6 伏笔 / L7 跨章跨卷 / L8 因果链 / L9 空间位置一致

**GATE-3**：13 维 + 读者视角 12 项 + L1–L9 全 PASS 且 logic_check.py（C1–C6/C8/C11/C18/C19/C20 无 FAIL；C7/C9/C10/C12/C13/C14/C15/C16/C17/C21/C22 的 WARN 已在读者视角审查 / L9 逐条显式回应）→ 进阶段四；否则带 FAIL 项回阶段二修订。

> 不变量：未跑 logic_check.py 机械校验（C1–C22）且无逐条自查结果 → 不提交。

## 五、去 AI 味专项清单（番茄技法本地化·补强文字 E 维度）

> 来源：番茄小说写作助手「去除 AI 味」技法，本地化为本作写作纪律。AI 默认用同一种腔调写所有场景，易堆砌修饰词/四字格/抽象陈述，读者可感出戏（E1/E3）。写章后逐条自查：

- **禁堆砌形容词**：璀璨/瑰丽/绚烂/磅礴/深邃 等"大词"单章 ≤2 处，且须有具体物象/动作支撑，不得空泛贴标签。
- **四字格律节制**：成语/四字短语单章 ≤6 处，避免连篇对仗堆砌（尤其章末钩子句不得用四字格收）。
- **抽象→具体**：凡"他很伤心/局势紧张/氛围肃杀"类抽象陈述，改由动作/对话/感官细节演出（show，非 tell；见读者视角 B4/#13 反向冗余）。
- **长短句交替**：连续 3 句以上同长度（尤其全为长句或全为短句）须打断，用短句切节奏、长句铺陈。
- **口语化声口**：对话须带人物声口特征（见 `IDX_PERSONA.md` §2），不得说明书式对答。

## 六、章末钩子类型库与轮换登记（番茄技法本地化）

> 既有「章末钩子强定义（截断/未知/危机）」+「相邻不重复」已覆盖基础；本节点补**类型库 + 逐章登记**，防连章同套路（读者视角 B2）。

**钩子类型库（≥10 种）**：①截断悬念 ②未知 ③危机 ④反转 ⑤抉择 ⑥情报差 ⑦关系突变 ⑧环境异变 ⑨时间压迫 ⑩疑问。

**轮换登记**：每章写后在大纲/自查报告登记所用钩子类型；**相邻 3 章不得用同一类型**（量化扫描核验，见 `docs/流程体系_小说创作.md` §3）；连章同类型视为节奏失控须回改（对标阶段三 #12 跨章防雷同）。
