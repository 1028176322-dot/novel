## §1 主线结构 + 黄金节奏点

### 1.1 主线句（一句话概括全书）
模板：**谁 + 在什么困境下 + 为了达成什么目标 + 必须对抗什么力量 + 最终获得或失去什么**。

本书主线句：
> 被遗弃时已是穿越者的国防大学轻武器研究生肖凡，在大晟王朝永熙年间为查清父母之死、带着自己人活下去，以「火器 + 玉佩修炼」两条线碾压江湖与庙堂，最终改写王朝与自身命运。

### 1.2 起承转合（功能占比，非平均分）
| 阶段 | 占比 | 本书对应 | 功能 |
|---|---|---|---|
| 起 | 前 10–15% | 卷一 1–100（0–16岁） | 立主角、立困境、抛大悬念（父母之死/玉佩来历） |
| 承 | 中 50–60% | 卷二 101–250 / 卷三 251–450 / 卷四 451–650 | 层层加码，阶段高潮递进 |
| 转 | 约 15–20% | 卷三末—卷四初（三皇子死、皇长孙被毒、听雨全面浮出） | 目标从「个人生存」转向「王朝/国运」 |
| 合 | 后 10–15% | 卷四 571–650（决战听雨 + 北境钩沉） | 收束沈括/父母/宗师等核心伏笔 |

### 1.3 黄金节奏点（每区间四要素）
每个区间必须提前写明：①阶段目标 ②爽点 ③小危机 ④读者期待。
字数区间灵活，但每个区间都要「有任务、有变化、有推进」。
本书落地方式：v2 大纲每章 6 节拍已含「爽点标记」与「章末钩子」，相当于把黄金节奏点细化到了章级。

## §2 升级体系

### 2.1 底层能源（1 主 1 次）
- 主：玉佩灵气（七境界修炼体系）
- 次：现代工科知识 / 悟性（火器研发、工业组织）

### 2.2 大阶数量（守住 4–7）
- 玉佩七境：凝灵 → 化精 → 淬体 → 易筋 → 洗髓 → 通神 → 归元（7 大阶，每阶可前中后期）
- 凡俗武道：铁皮 → 铜骨 → 金身/气感 → 周天 → 化罡（5 大阶）
- 双体系并存，但需明确「为什么不乱升」：玉佩修炼靠玉佩资源稀缺，凡俗武道靠资质与功法封锁。

### 2.3 每阶质变内容（不能只涨数值）
| 大阶 | 质变 |
|---|---|
| 铁皮/铜骨 | 身体硬抗刀剑，近战肉搏 |
| 金身/气感 | 真气雏形，近身可战更强 |
| 周天/化罡 | 内力外放，宗师级 |
| 易筋（卷三） | 肉身强韧如筋索，越阶战基础 |
| 洗髓（卷四） | 洗髓伐骨，寿元与承载力跃升 |
| 化罡圆满（卷四） | 宗师门槛，可正面接火器外武者 |
| 通神/归元（卷五+） | 法则/世界级权限，驻颜长生 |

### 2.4 体系与世界结构绑定
- 高阶资源（功法、灵矿、火器机密）由朝廷/宗门/听雨阁垄断；低阶散修/军士靠军功、门派贡献。
- 普通人天花板：铜骨为军中精锐主流，化罡为宗师级；玉佩修炼者同阶可战更高一层，但每阶突破资源稀缺。

### 2.5 例外机制（肖凡 = 双核心，四列必填）
| 项 | 内容 |
|---|---|
| 来源 | 穿越者记忆 + 玉佩 |
| 优势 | 火器降维打击 + 灵气嫁接 |
| 代价 | 工业链依赖、研发周期、原料、政治风险 |
| 边界 | R4（火器对宗师基本无效，狙击枪卷四例外 FB-065）；R5 时间锁（无烟火药/狙击枪卷四才解锁） |

### 2.6 每阶输入—过程—输出（待补全到 IDX_SETTING）
- 例：易筋 → 输入：玉佩灵气 + 高压战斗；过程：灵气淬炼筋骨如索；输出：肉身越阶抗铜骨、近身可战。
- 洗髓 / 化罡圆满 等需在设定索引补全同结构。

### 2.7 资源稀缺度表（待补）
| 层级 | 资源 | 获取渠道 | 使用门槛 |
|---|---|---|---|
| 低阶 | 铁料、黑火药 | 民间/工坊可控 | 手艺 |
| 中阶 | 膛线、定装弹 | 朝廷/工坊技术壁垒 | 班底+专利 |
| 高阶 | 无烟火药、狙击枪、玉佩灵物 | 肖凡核心班底 + 内卫府双重把控 | 核心权限 |

## §3 基础世界观

### 3.1 六要素
| 要素 | 本书现状 |
|---|---|
| 时空背景 | 大晟王朝永熙年间（架空仿明），1370=永熙三年基准；主舞台：盛京/清虚观/南州/北境/江湖 |
| 社会结构 | 上层皇室/国师/重臣；中层世家/官宦/江湖门派；底层平民/工匠/流民；边缘乞丐/谍子/遗孤 |
| 运行规则 | 生存（手艺/军功/田产）、竞争（武籍/功名/资源）、交易（市井+情报网）、惩罚（武者登记新政/谋反论） |
| 核心矛盾 | 皇权 vs 世家/江湖；技术碾压 vs 传统武力；个人生存 vs 王朝利益 |
| 超凡系统 | 玉佩七境 + 凡俗武道；力量来源/等级/代价/对社会影响（R2/R4/R5 锁规则） |
| 主角入场 | 被遗弃婴儿/遗孤 → 底层求生 → 被迫卷入国战 |

### 3.2 撰写思路（已落地）
定主舞台（盛京+江湖）→ 定 3 条核心规则（武者登记/皇权-江湖制衡/火器垄断）→ 定 2–3 组对立（皇权vs江湖、技术vs武力、个人vs王朝）→ 定入场（遗孤求生）→ 先立核心再扩外层（卷五/卷六长生闭关时间跳待重锚）。

## §4 配角与反派

### 4.1 反派自洽逻辑（非工具人）
| 反派 | 自洽逻辑 | 与主角冲突落点 | 退不了的理由 | 局部正确性 |
|---|---|---|---|---|
| 三皇子萧沛（已死） | 夺嫡 + 与听雨阁合作构陷大皇子 | 皇权正统 vs 肖凡卷入 | 已死，遗策由裴琰承 | 维护自身血脉继承权 |
| 裴琰 | 萧沛余党，执行遗策 | 利益/权力 | 萧沛死，出路只剩复立旧势力 | 报知遇 |
| 听雨阁（虞国） | 国家级间谍机构渗透大晟 | 国战利益 vs 肖凡火器改北境战局 | 机构使命，退则失据 | 保虞国国运 |
| 顾衡（卷四 FB-073 真凶） | 沈括灭门真凶 | 真相/复仇 | ⚠ 待补「他为何必须灭门」 | 自认护己方势力 |
| 言先生 | 听雨江南分舵主，远遁非灭 | 组织任务 vs 肖凡追查 | 身份暴露即死 | 职业间谍的忠诚 |

### 4.2 配角独立目标（非工具人）
老柯（手艺被认可+跟肖凡搞出真火器）、苏墨凝（查清父案+护母）、玄尘子（国师地位+道门/皇权平衡）、青溪派（宗门存续站队）。

### 4.3 人物冲突关系网（四问）
① 核心诉求 ② 底层动因 ③ 与主角冲突落点 ④ 多边制衡/结盟/反目。
示例：苏墨凝（诉求：查父案）vs 青溪派（旧怨暗线）→ 主角夹在中间，关系复杂化（待在正文强化）。

## §6 如何把故事写长/写丰富

> 本模块在 `写作规则.md` §五「情节丰富三要素」「环境/氛围描写」「节奏张弛」已**高覆盖**，此处仅补要点与差异。

- 情节三幕：前置铺垫（FB 预埋已系统化）→ 增添过程（写尝试/失败/挣扎，非结果陈述）→ 延续余波（情绪/关系/隐患/新矛盾联动，章末钩子已做）。
- 氛围：环境贴情绪与题材，玄幻重天地异象/宗门景象，超 100 字风景自查水文。
- 节奏张弛：高潮细化 + 穿插轻量日常 + 适度反转 + 适当延伸 + 深挖可能性（身世/配角暗线/好坏灰度）。
- 避坑：不写无关内容 / 不重复冗余 / 不脱离人物逻辑；丰富 ≠ 拖沓。

## §7 战斗节奏

- 分层：试探 → 交手 → 转折 → 爆发 → 收尾。
- 节奏公式：信息差 → 压制 → 反转 → 碾压/惨胜。
- 越阶战频率严控（R4/R5）：火器对宗师基本无效，狙击枪 FB-065 是卷四唯一例外，破宗师后不可常态化。
- 多线编排：战场 / 暗线 / 朝堂同步，交汇处放大爽感。
- 爽感释放点 = 反派认知被颠覆的瞬间（「铁管子」→ 宗师倒下）。

---

## §8 情绪曲线

- 每章情绪弧：平静/压抑（10%）→ 紧张（50%）→ 峰值爽点/危机（30%）→ 余波（10%）。
- 宏观卷级起伏：卷四五幕即一条大曲线（压→升→峰值→对撞→余波+新钩）。
- 压抑-释放周期：先压后放，每 1–2 章给小回报（黄金节奏点）。
- 情绪管理：恐惧（班底安危）→ 期待（技术突破）→ 满足（反派吃瘪）交替。

## §9 文笔风格

- 工业党写实（R19）：火器写真实工艺，不糊弄。
- 展示不讲述：用动作/细节代形容词（写作规则 §五 抽象词替换、身体语言）。
- 对话贴人设（R20）：老柯糙话、苏墨凝含蓄、玄尘子云山雾罩；穿越者内心戏独白。
- 动词驱动、短句为主；环境贴情绪（呼应 §6）。
- 避免水文（写作规则 §七 避坑）。

## §10 男频爽文节奏公式

| 层级 | 频率 | 本书落点 |
|---|---|---|
| 小爽点 | 每 1–2 章 | 技术小突破 / 一个轻视者被噎 |
| 中爽点 | 每 3–5 章 | 以技破武碾压战 |
| 大高潮 | 每 8–12 章 | 幕内转折（狙击枪首杀宗师） |
| 卷终决战 | 每 15–20 章 | 卷四 571–650 决战听雨 + 北境钩沉 |

四大放大器：① 压抑前置 + 即时回报 ② 信息差爽点（主角/读者知、反派不知）③ 扮猪吃虎（肖凡文弱样）④ 数值化成长反馈（等级/装备/战绩可视）。

## §12 使用方式

1. **写大纲前**：按 §1.4 模板表填卷级主线；按 OUTLINE_FORMAT §5 SOP 查三索引。
2. **写正文前**：读 写作规则.md（尤其 §五/§七/§十二/§十八）+ 本章 v2 节拍；执行 10 条自查。
3. **查漏**：对照 `方法论_查漏补缺清单.md` 的「待补项」，把新规则补进 写作规则.md 或正文规范。
4. **复盘**：每卷完稿后，用 §1.4 模板表 + §8 情绪曲线回看节奏是否断档。
