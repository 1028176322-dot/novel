# 治理文档版本清单（GOV_VERSION）

> 统一管理各治理文件的版本戳与覆盖范围，配合 `validate_governance.py` 做一致性校验。
> 版本戳格式：`YYYY-MM-DD 覆盖 X–Y 章`。任何治理文件重大更新须同步更新本清单 + 该文件顶部版本戳行。

## 维护约定
- **单文件全量覆盖，不按卷拆分**：五张 IDX_*.md 均为 1–1000 章单文件全量。理由——按卷拆分成 6×N 个小文件会放大「跨卷引用 / 回填遗漏 / 路径漂移」风险，且机械校验（`logic_check.py`）与 `validate_governance.py` 均按单文件全量扫描设计；若未来 1000 章规模下检索变慢，再评估按卷拆 + 符号链接，不现在做。
- **版本戳**：每个 IDX_*.md 顶部第二行 `> 版本戳：…`。
- **变更记录**：统一在 `AUTHORITY_INDEX.md` 末尾「变更记录」追加，不各文件自记流水。
