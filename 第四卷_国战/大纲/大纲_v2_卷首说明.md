## 4. 卷四技术线 / 修炼线演进（红线 R5 + 总体规划表）

### 4.1 科技线（R5 严守：卷4 = 无烟火药 + 狙击枪）
- **起点**（卷三末）：后装枪量产（FB-048，黑火药）+ 灵气火器雏形（FB-049，黑火药基底灵气导引）。
- **卷四演进**：
  - FB-064 **无烟火药研发**：硝化棉/硝化甘油类（基于已有黑火药体系升级，肖凡配方，非原生）。射速/射程/精度/存储全面提升，取代黑火药前装/后装弹药。R5 卷四解锁，守 R2（仍肖凡发现）。
  - FB-065 **狙击枪研发**：线膛高精度 + 无烟火药 + 简易瞄具（机械/光学雏形），对宗师致命（解 FB-042/R4/R6）。卷四科技顶，战场支配力。
  - FB-068 **以技破武·国战**：狙击枪对虞国武者/宗师战场碾压，建立"技破武·国战"声望（贯穿全卷）。
  - FB-048 **后装枪量产上战场**：卷三已成，卷四量产供给边军（仍黑火药，逐步被无烟火药替代前沿）。
  - FB-049 **灵气火器战场化**：卷三雏形，卷四用于战场规模化（仍黑火药基底/灵气导引，守 R2；卷五成熟）。
- **禁用**：卷五方成熟的灵气火器大成（卷四仅战场化，不突破卷五成熟线）；不得出现超越卷四科技层级的武器。

### 4.2 修炼线（总体规划表：卷四 = 化罡圆满 / 七境界 易筋→洗髓）
- FB-066 **洗髓**：七境界·洗髓（易筋后阶，卷四修炼顶，肉身重塑如髓洗，越阶战基础）。
- FB-067 **化罡圆满**：凡俗武道内力顶（周天→化罡→圆满），罡气外放成势。
- 卷三已达：易筋（FB-053）/ 金身圆满（FB-054）。卷四由易筋圆满推进至洗髓，化罡圆满。
- **代价机制**（设定总览 v1 §3.3）：每境有心魔劫，玉佩已碎无补救；突破需灵气环境/天材地宝。卷四推进至洗髓，长生（FB-005）仍为远途（卷五）。

### 4.3 班底演进（R7）
- 既有：老柯（管造 FB-013）、沈遇（管账 FB-017）、苏墨凝（暗线 FB-023）、卷三江湖班底（FB-063：莫铁/窦九/钱小乙/文仲/严九/陆九/小满/铁牛等）。
- 卷四扩：FB-076 **战争班底**（军械/参谋/斥候/边军联络，支撑国战产业化与情报网）。

### 4.4 暗线演进（承接卷三）
- FB-008 **沈括灭门**：卷三余波线索移交国师，卷四揭露幕后真凶（FB-073，非二当家）。
- FB-012/FB-057 **听雨阁**：卷三远遁非灭，卷四方全面对抗（FB-072），破主力。
- FB-019 **青溪派**：卷三站队，卷四作为江湖外援随肖凡入国战体系。
- FB-010 **四皇子**：卷三萧俊稳；卷四萧恒被构陷（FB-069）、萧厉被挑火并（FB-070）——皇权火并高潮（萧沛已死禁复活，遗策由余党裴琰承线）。
- FB-024 **国师**：卷三国战托底，卷四颁国战令、以军籍护肖凡。
- FB-055 **黑虎帮幕后**：卷三牵出，卷四揭露真凶（FB-073）。
