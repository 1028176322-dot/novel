# 《道法百年》小说大纲 v1.3

> ⚠️ **本文档为 v1.3 旧总纲，已废弃。** 时间线与年龄以大纲 v2 各段文件及《时间线与人物关系图.md》为准（当前权威：肖凡 **15 岁自永熙十八年(1385) 起**，第21–60章均为 15 岁；旧框架「第41章14岁回观、第50章15岁结束」已作废）。

> *一个穿越者，带着现代知识和神秘功法，在大晟王朝活过三百年的故事。*

**写作规范：每章3000字左右**

## 一、核心设定

### 基础信息

| 项目 | 设定 |
|------|------|
| **主角** | 肖凡，1370年出生，国防大学轻武器方向研究生，被遗弃时已是穿越者，携带完整现代记忆 |
| **金手指** | 神秘功法，唯一；每80年修炼期满沉睡10年（可调面容）→ 返老还童 |
| **返老还童后** | 身体少年，心境/记忆/内功保留；通过忠心之人 + 现代毒药重掌势力 |
| **战力体系** | 参考金庸武侠：修炼时间累积内力，天花板≈张三丰 |
| **历史设定** | 完全虚构的大晟王朝，金庸武侠体系，无玄幻；历史走向不绑正史 |
| **争霸结局** | 另立新朝（近现代化国家） |

### 入仕路线

文武双线交替——明面文官掌朝堂，暗中掌控军权。

### 情感线

遗憾（错过或生死相隔）。

- **位置**：深山中，隐秘

## 四、第一周期分卷大纲

### 第一卷【道生】0-15岁（1370-1385）

**时间：1370-1385年，共50章**
**核心主题：立足——找到方向，埋下种子**

#### 章节大纲

##### 童年篇（第1-2章）

**第1章 遗弃**
- 永熙三年（1370年），风雪夜
- 婴儿被遗弃于清虚观门口
- 无为子发现，抱入观中，赐名肖凡
- 襁褓中有一块玉佩（伏笔：与功法有关）

**第2章 童年·道心**
- 肖凡在道观成长，接受现实
- 四位师兄轮流照顾
- 无为子亲自教导：识字、读经、基础道家功法
- 肖凡表现异乎寻常的早慧

##### 筑基篇（第3-20章）

**第3章 后山**
- 肖凡6岁，玉佩指引发现后山隐蔽山洞
- 石壁刻有完整功法
- 主角意识到：这是他在这个世界活下去的资本
- 第一次感受气感，踏入修炼之路

**第4-7章 功法入门**
- 修炼入门，心无杂念，气沉丹田
- 用现代思维理解功法逻辑
- 功法理解逐步深化
- 修炼过程中与师兄们的日常互动

**第8-11章 师兄情谊**
- 与四位师兄的感情建立
- 大师兄守一：稳重可靠，帮无为子管理道观
- 二师兄问玄：聪明伶俐，已在山下集市做小买卖
- 三师兄铁云：武功最好，力大无穷，想下山闯荡
- 四师兄清思：痴迷读书，与肖凡最聊得来
- 肖凡暗中压制实力，不显山露水

**第12章 灭门**
- 沈府灭门案发，沈括满门四十七口被黑虎帮残党灭门
- 驿卒带急报往州府狂奔
- 伏笔：此事日后成为肖凡游历时的一桩悬案

**第13章 朝议**
- 驿卒到州府，州府震怒，派差役赶往现场
- 现场只剩灰烬，确认是江湖手段
- 州府上报省府，八百里加急送盛京
- 萧咤在朝会上得知，震怒
- 朝议：武者不受管控，议定清查江湖门派，武者需登记在册
- 结尾：二师兄林谋下山置办物资，看到街头张贴的武者登记告示，回山告知无为子

**第14章 新风**
- 永熙十三年春，肖凡10岁（注：第13章写朝堂线，肖凡年龄跳过9岁）
- 清虚观收到官府文告，无为子召集弟子告知朝廷新政
- 肖凡第一次意识到"朝廷"的存在
- 修炼：冲玉枕关

**第15章 初下**
- 玉枕关通，小周天彻底顺畅
- 无为子第一次带肖凡下山办事，让肖凡见识山下世界
- 城里贴满通缉黑虎帮二当家的文告

**第16章 江湖**
- 随四师兄宋明入山采药，途中碰到一伙江湖人
- 宋明低调避开，肖凡第一次近距离感受到"江湖"
- 听到酒馆里有人议论朝廷清查武者的新政

**第17章 微热**
- 在后山洞窟修炼时，玉佩突然微微发热
- 肖凡以为是走火入魔的征兆，把玉佩摘下来观察
- 温度又恢复正常，去找无为子，无为子只说"戴着就好，不必多问"
- 伏笔：玉佩未激发，只是预警

**第18章 处世**
- 无为子开始教肖凡"下山后如何自处"——处世之道
- 朝廷新政之下，江湖人身份敏感，如何不惹事、不惹官、不暴露修为
- 肖凡问："那师父让我下山，是因为观里待不下去了么？"
- 无为子笑："是你自己该走了，观里能教你的，已经教完了。"

**第19章 切磋**
- 与四位师兄逐一切磋，检验修炼成果
- 玉枕关已通，小周天顺畅，内力在小腹处开始积蓄（下丹田）
- 无为子看完，点头："开春后，我带你下山。"
- 肖凡却说："弟子想再练一年，把下丹田的积蓄再厚一些。"
- 无为子没反对

**第20章 除夕**
- 永熙十三年冬末，肖凡10岁
- 师徒五人在观里过年
- 无为子讲起自己年轻时的游历故事
- 肖凡听着，心里开始盼着开春
- 第一卷筑基篇收尾

##### 游历篇（第21-40章）

**第21章 出发**
- 永熙十四年春（1381年），肖凡11岁（注：第20章结尾肖凡10岁，开春后11岁）
- 无为子带队下山游历
- 师傅开导：*道法自然，道在人间，不在山上。*

**第22-25章 人间**
- 亲眼看到战乱后的人间：荒村、难民
- 第一次感受到世间的残酷
- 见识大晟王朝基层政治生态
- 观察科举制度下的众生相

**第26-30章 城市**
- 见城镇繁华与贫民窟的对比
- 肖凡开始思考社会问题的根源
- 城里仍在通缉黑虎帮二当家，但毫无线索
- 朝廷清查武者的新政在各地执行力度不一

**第31-35章 江湖**
- 见识江湖人士和门派
- 发现军户制度崩溃、武器落后的问题
- 与师兄们在游历中加深羁绊
- 主角一边跟随游历，一边暗中修炼，玉佩偶尔微热

**第36-40章 积累**
- 火药改良在游历途中悄悄完成
- 主角观察能力提升，学会隐藏自己
- 见识各地风土人情，社会经验积累
- 四年游历即将结束，决定返回道观

##### 归来篇（第41-50章）

**第41章 归来**
- 永熙十七年（1384年），肖凡14岁，游历结束返回清虚观
- 四年游历归来，五兄弟感慨万千
- 主角开始认真规划未来方向

**第42-43章 火药**
- 肖凡14岁，在后山秘密实验，火药改良最终成功
- 爆炸力提升约30%
- 制作第一个"震天雷"（原始手榴弹）
- 确认：现代知识+这个世界=可行

**第44-46章 风雨**
- 无为子病倒（多年积累的衰老）
- 遗言（洒脱风格）：*道法自然，道在人间，不在山上。*
- 将铜牌交给肖凡
- 无为子仙逝，肖凡在师父墓前发誓

**第47-49章 玉佩**
- 师父亡后，肖凡准备独自下山
- 玉佩在师父下葬当晚突然发热、发光
- 完整功法直接灌入肖凡大脑
- 玉佩随即粉碎——金手指正式激活
- 肖凡内力暴涨，但外表仍是少年

**第50章 下山**
- 永熙十八年（1385年），肖凡15岁，独自下山
- 师兄们留在道观待命
- 约定：将来需要时，请师兄们出山相助
- 第一卷终——*一个新的起点*
- 主角迈向朝堂，心中有了明确的方向

### 第二卷【江湖】15-25岁（1385-1395）

**时间：1385-1395年**
**核心主题：游历天下，了解世界，建立人脉**

#### 章节大纲（草案）

**第1章 游学**
- 肖凡以学子身份游历各方
- 三年之约临近，先去探望师兄们

**第2章 问玄的铺子**
- 拜访二师兄问玄
- 问玄已在盛京开了杂货铺
- 见识问玄的商业手段和情报收集能力
- 情报网初见雏形

**第3章 铁云的江湖**
- 拜访三师兄铁云
- 铁云拜入小门派，成了记名弟子
- 见识江湖门派纷争
- 铁云引荐肖凡认识几位江湖人士

**第4章 清思的旅途**
- 与四师兄清思会合
- 清思在游历中实验火药，失败三次但有进展
- 两人深入交流现代知识
- 科技线思路进一步清晰

**第5章 军制之殇**
- 肖凡接触边军/地方武装
- 发现大晟初年军制腐败：吃空饷、训练废弛、武器落后
- 用现代知识记录问题
- 军事改革念头萌芽

**第6章 永熙晚年**
- 功臣被清洗，朝堂风声鹤唳
- 主角观察局势，判断大晟王朝政治走向
- 意识到朝堂凶险，决定先稳固根基

## 九、毒药体系详细设定

主角的现代化学知识不仅是火器研发的利器，更是控制势力的"安全锁"。这套毒药体系是主角沉睡后重掌势力的核心手段之一。

### 一、核心原理

| 原理 | 说明 |
|------|------|
| **理论基础** | 现代化学知识（生物碱、重金属、有机化合物等）|
| **制作材料** | 利用大晟王朝已知的药材/矿物，提取、合成 |
| **核心逻辑** | 毒药≠杀人，而是"定时炸弹"——平时无害，定期服解药则安全 |
| **特点** | 无色无味，发作缓慢，症状像普通疾病，难以被普通大夫诊断 |

---

### 二、毒药分类

#### ① 核心控制毒（最常用）

| 名称 | 成分 | 效果 | 解药 |
|------|------|------|------|
| **锁心散** | 铅丹+朱砂+特定草药 | 服后无感，每月需服解药一次，否则心脏逐渐衰竭 | 绿色药丸，每月一颗 |
| **断脉丸** | 乌头碱稀释液 | 阻断经脉运行，不服解药则内力逐渐消退 | 棕色药粉，溶于酒中服用 |
| **噬骨粉** | 汞化合物 | 侵蚀骨骼，不解则骨痛难忍，最终瘫痪 | 银色药丸，每周一颗 |

#### ② 高级情报毒（针对虞朝渗透）

| 名称 | 效果 | 用途 |
|------|------|------|
| **忘川水** | 让人说出真话，但事后记忆模糊 | 审讯/辨别敌友 |
| **傀儡香** | 服后思维混乱，任人摆布，持续三个时辰 | 控制虞朝安插的棋子 |
| **七日断肠散** | 七日后必死，但每日有解药压制 | 紧急情况下的最后手段 |

#### ③ 自保毒（主角专用）

| 名称 | 效果 | 设计目的 |
|------|------|----------|
| **假死丸** | 服后呼吸心跳全无，持续三日 | 逃脱/布局假死 |
| **解百毒膏** | 剧毒，但能压制其他所有毒 | 万不得已时的保命手段 |

---

### 三、毒药体系的运作机制

#### 层级管理

```
主角（唯一掌握全部配方和解药）
    ↓
┌─────────────────────────────────────┐
│ 第一层：核心心腹（4-8人）            │
│ - 知道毒的名称，但不知完整配方       │
│ - 每月领取解药                      │
│ - 可以选择不服，但三年内必死        │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 第二层：外围势力（约30-50人）        │
│ - 不知毒的存在，只知每月要"领药"    │
│ - 被告知是"强身健体丸"             │
│ - 解药按季度发放                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 第三层：普通成员（数百人）           │
│ - 不知毒的存在                      │
│ - 只有核心心腹知道他们的存在        │
│ - 通过核心心腹间接控制              │
└─────────────────────────────────────┘
```

#### 分发机制

| 场景 | 方式 |
|------|------|
| **日常分发** | 核心心腹按月领取解药，名义上是"帮主配发的修炼丹" |
| **紧急情况** | 主角通过暗号/信物确认身份后，发放临时解药 |
| **叛徒处置** | 停止发放解药，观察反应，决定是给机会还是彻底清除 |
| **新成员加入** | 观察期后，视忠诚度决定是否用毒控制 |

### 四、毒药与势力的结合

#### 1. 沉睡前的布局（第四卷）

| 时间点 | 行动 |
|--------|------|
| 沉睡前20年 | 开始系统性研究毒药配方 |
| 沉睡前10年 | 在核心成员中逐步推广 |
| 沉睡前5年 | 建立毒药分发网络和暗号体系 |
| 沉睡前1年 | 将配方拆分为"毒方"和"解方"，分别交给不同的人保管 |
| 沉睡前 | 公布规则：背叛=死亡，服从=安全 |

#### 2. 苏醒后的重掌

| 步骤 | 操作 |
|------|------|
| **第一步** | 以信物/暗号确认身份（只有核心心腹知道）|
| **第二步** | 发放解药，证明身份真实性 |
| **第三步** | 观察势力运转正常，暗中接管 |
| **第四步** | 处理背叛者（停解药/清理）|
| **第五步** | 发展新成员，补充毒药体系 |

#### 3. 叛徒处理流程

```
发现叛徒迹象
    ↓
停止发放解药（观察一个月）
    ↓
叛徒出现症状
    ↓
┌─────────────────────────────────────┐
│ 情况A：主动认错求饶                  │
│ → 给予最后一次机会，但降级控制       │
├─────────────────────────────────────┤
│ 情况B：负隅顽抗                      │
│ → 彻底清除，以绝后患                 │
├─────────────────────────────────────┤
│ 情况C：逃跑到虞朝/蛮国              │
│ → 追杀，不留后患                     │
└─────────────────────────────────────┘
```

### 五、现代知识的应用

主角的优势不仅是"知道毒药配方"，而是"理解毒药原理"：

| 方面 | 优势 |
|------|------|
| **诊断反制** | 知道毒药如何发作，能伪装症状骗过外人 |
| **解药改良** | 能研发出比传统毒药更稳定的解药 |
| **快速解毒** | 危急时刻能紧急合成临时解药 |
| **新毒研发** | 不受限于传统配方，能根据需要创造新毒 |
| **检测识别** | 能通过现代化学知识识别别人下的毒 |

### 六、道德边界设计

毒药体系是双刃剑，主角需要把握分寸：

| 原则 | 说明 |
|------|------|
| **不滥杀** | 毒药只用于控制，不用于攻击（那是内力/火器的事）|
| **给机会** | 发现背叛后，先警告，再观察，给改正机会 |
| **有底线** | 对真正的底层成员不用毒，只用利益和感情 |
| **自省线** | 主角内心有过挣扎：这和虞朝的控制有何不同？ |
