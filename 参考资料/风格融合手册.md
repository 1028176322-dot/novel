# 风格融合手册（Style Fusion Handbook）

> **用途**：用户持续提供外部小说，AI 提取其写作风格，**有用点沉淀至此手册**，并在《道士也玩火力覆盖》章节写作中调用，充实正文质量。
> **性质**：本手册是「外部风格 → 本作笔法」的**唯一累积源**。每提取一部新小说，向其对应文件（如 `庆余年风格提取与仿写.md` / `赘婿风格提取.md`）追加细节，并把**可落地的笔法规则**合并进本手册「一、已沉淀笔法」。
> **调用时机**：写章前（阶段一/阶段二）consult 本手册取笔法增强；阶段三 #1 文风一致性须对齐本手册。
> **兜底红线**：一切外来笔法不得突破 `写作规则.md` 红线 R1–R21、R14 客观呈现、R17 藏拙、R19 零现代词（台词/旁白，经营理念现代词例外仅限肖凡·详见 `IDX_PERSONA.md` §1 锁词表）、年号永熙≤37（C3）、空间锚点含清虚观/盛京等（C7a）。

## 一、已沉淀笔法（按维度，持续累积）

> 每条格式：**技法 · 来源 · 本作迁移 · 红线注意**

### 1. 叙事视角与节奏
- **限知 + 史笔双层**〔庆余年·赘婿〕：章内第三人称限知锚定肖凡；章首/转场用「永熙X年，大晟……」史笔旁白定调时代（年号≤37）。宏观背景压微观日常。
- **慢热日常密度**〔赘婿〕：把「过日子」写足，人物与关系在琐碎中生长；事件密度低、心理/关系密度高。肖凡线清虚观日常（扫阶、炼体、与师兄互动）可铺足。
- **多线硬切场**〔赘婿〕：多条人物线（师父/香客/林谋/宋明）用 `*********` 或空行硬切，不靠过渡段软转（本作 md 不强制，但转场须有锚点）。
- **碎片记忆蒙太奇**〔赘婿〕：战斗/受创用「一片破碎的记忆……」碎片拼贴 + 俯瞰视角，不直叙全貌；临场眩晕感强。**不写火器构造/膛压/配方**（R1–R3）。
- **信息 withholding**〔庆余年·赘婿〕：用 `……` 省略号、未竟对话、暗线伏笔制造悬念，不一次交底。

### 2. 人物塑造
- **白描定调**〔庆余年·赘婿〕：外貌一两笔抓特征与反差（瓜子脸/乌黑长发/苍白的手），不堆砌。
- **专业视角转译**〔赘婿〕：肖凡用「前世所受训练与阅历」冷静解剖人际（看人一眼便知轻重、出手有章法），文中只显「老成得不像九岁」「像是早经大事」，**不点名现代科技/品牌/网络/影视**（R1–R3、R14、R17）。
- **个性化声口**〔庆余年·赘婿〕：一人一声口，绝混。肖凡冷静克制偶带冷面幽默；师兄各依性格（陈守沉稳/林谋机敏/周猛直爽/宋明温润书痴）。
- **小动作显权位/性格**〔赘婿〕：一个眼神一个动作定尊卑、透城府（肖凡藏拙的「收着力」即此类）。
- **反差人设**〔庆余年·赘婿〕：卑微皮囊 + 凌厉内核 → 肖凡「孩童躯壳 + 沉静心性/前世阅历」。

### 3. 环境与场景
- **白描 + 感官器物**〔庆余年·赘婿〕：用器物+感官写场景（扫阶声、老槐影、丹炉火、宋明的药筐、炭炉、破伞）。环境服务心境。
- **史笔锚时代**〔庆余年·赘婿〕：清虚观/盛京/下山脚夫路——须含 KNOWN_LOCS（C7a）。
- **市井/园林精描**〔赘婿〕：集镇烟火气、道观宁静古朴，空间有呼吸感。

### 4. 语言风格
- **半文半白**〔庆余年·赘婿·本作原有〕：古风对话（相公/福身 等改本作古风）+ 现代流畅叙事；现代词仅限肖凡内心（R19）。
- **长句析理 + 短句对话**〔赘婿〕：分析段绵长复句；对话短促利落。
- **生活化比喻 + 冷幽默**〔庆余年·赘婿〕：取生活/训练意象比喻（如书卷如眉黛、铁骑如月牙）；冷面幽默、反讽。
- **`……` 留白**〔庆余年·赘婿〕：省略号高频，表留白/语气/未竟。
- **史笔议论**〔赘婿〕：旁观者/史家口吻评时代与人性，苍凉克制 → 化用成本作「永熙X年，大晟……」旁白。**不带作者碎碎念/求票/现代口语梗**（网文互动体例，勿迁移）。

### 5. 对话风格
- **神态动作标签密**〔赘婿〕：每句对白带神态/动作标签（撇撇嘴/冷冷一笑/咬了咬牙），动作即情绪。
- **短平快 + 潜台词**〔庆余年·赘婿〕：信息藏斗嘴与潜台词，少直球；不打直球（情感/意图多曲笔）。
- **冷面打趣**〔庆余年·赘婿〕：肖凡式一本正经开玩笑、逻辑碾压（守古风、不泄现代），对应宁毅/范闲灵动。
- **权谋机锋**〔赘婿〕：表面客气、暗藏试探与威慑（师父史笔式告诫、朝廷线对话可取法）。

### 6. 情感与氛围
- **冷峻底色 + 温情闪现**〔庆余年·赘婿〕：温情落点放师父/宋明/小动物等；借细节传递（老槐影、扫阶声、丹炉火），不直抒。
- **家国/时代悲怆**〔赘婿〕：苍凉感落「永熙十二年，江湖人将入笼」式史笔旁白。
- **留白悬念**〔庆余年·赘婿〕：用「无人知晓/暗线/破碎记忆」铺陈，情感不一次倾泻。
- **反差张力**〔庆余年·赘婿〕：悠闲日常 vs 山雨欲来（清虚观之静 vs 沈括灭门/武者登记之暗涌）。

## 三、Do / Don't 速查

**Do**
- 限知锚定肖凡；章首/转场史笔旁白「永熙X年，大晟……」定调。
- 用器物+感官写清虚观日常（扫阶声、老槐影、丹炉火）。
- 对话必带神态/动作标签，短平快、藏潜台词。
- 肖凡「老成」以细节显，不解释来源（前世阅历转译）。
- 受创/冲突用碎片蒙太奇 + 俯瞰，留白。
- 温情借细节（师父、宋明、小动物）不抒情。

**Don't**
- 不点名现代科技/网络/影视（R1–R3、R14、R17、R19）；现代实物名词/网络梗仍禁——**经营理念现代词（品牌/口碑/供应链等）仅限肖凡按 `IDX_PERSONA.md` §1 锁词表放行**，不在此禁列。
- 不写火器构造、膛压、硝石配方（R1–R3）。
- 不带作者碎碎念、求票、现代口语梗。
- 年号不超 37（C3）；空间须含清虚观/盛京（C7a）。
- 不泄「重生/穿越」字眼硬解，以「前世阅历/所受训练」含蓄。
- 不替读者解说（show don't tell，禁反向冗余复述）。
