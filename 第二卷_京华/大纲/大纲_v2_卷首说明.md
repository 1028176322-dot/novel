## 4. 卷二技术线 / 修炼线演进（红线 R5 + 总体规划表）

### 4.1 科技线（R5 严守：卷2 = 膛线 + 定装弹药）
- **起点**（卷一末）：燧发枪雏形（FB-016 枪管 + FB-018 颗粒火药）。
- **卷二演进**：
  - FB-026 **膛线工艺**：手工拉膛/缠膛，提升精度与有效射程（仍黑火药，非线膛步枪级，受工艺限制）。
  - FB-027 **定装弹药**：纸壳定装（火药+弹丸一体），提升射速与战场可靠性；为"火器产业化"关键。
  - FB-028 **火器作坊产业化**：盛京工坊量产体系（老柯管造、招匠师），告别卷一"手工单支"。
- **禁用**：后装枪（卷3）、无烟火药+狙击枪（卷4）——卷二不得出现。

### 4.2 修炼线（总体规划表：卷二 = 铜骨→金身 / 周天→化罡 / 七境界凝灵→化精）
- FB-033 **金身突破**：铜骨→金身（横练三阶顶，寻常兵刃难伤，需内力持续滋养）。
- FB-034 **化罡突破**：周天→化罡（内力外放成罡气）。
- FB-035 **七境界·凝灵**：引灵气入体（玉佩功法，肖凡独有）。
- FB-036 **七境界·化精**：灵气化精，滋养横练暗伤（与金身突破互为支撑）。
- **代价机制**（设定总览 v1 §3.3）：每境有"心魔劫"，心性不稳则走火入魔；玉佩已碎，练错无补救；突破需灵气环境/天材地宝（稀缺）。卷二仅推进至化精，长生（FB-005）仍为远途。
- FB-042 **火器对宗师实测**（R6 推进）：卷二借一次宗师级对峙，确证"膛线定装火器对宗师基本无效"，为卷四狙击枪埋线。

### 4.3 班底演进（R7）
- 既有：老柯（管造 FB-013）、沈遇（管账 FB-017）、苏墨凝（暗线 FB-023）、两护院。
- 卷二扩：FB-037 新增班底（匠师/账房/护院），支撑产业化与情报网。
