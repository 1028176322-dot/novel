## 一、时间线

```
永熙三年（1370年）冬
│
├─ 肖凡出生/穿越（前世：国防大学轻武器研究生）
├─ 被遗弃在清虚观门口（盛京以南三百里群山深处）
├─ 襁褓中携带：玉佩、几枚铜钱
├─ 无为子（73岁）收养，命名"肖凡"
└─ 第1章《遗弃》

永熙三年→八年（1370-1375年）
│
└─ 肖凡成长，逐步确认自己穿越到大晟王朝（架空），皇帝萧咤

永熙八年（1375年）冬
│
├─ 肖凡5岁
├─ 无为子将玉佩正式交给肖凡
├─ 肖凡触摸玉佩，触发前世记忆片段（地摊买玉佩）
├─ 介绍四位师兄背景
├─ 早课日常、晚饭日常
└─ 第2章《童年·道心》

永熙九年（1376年）夏
│
├─ 肖凡6岁
├─ 玉佩到手，尝试各种方法激活：
│   ├─ 滴血 → 无反应
│   ├─ 火烧 → 无反应
│   ├─ 水泡 → 无反应
│   ├─ 暗语（"芝麻开门"） → 无反应
│   └─ 意念感知 → 无反应
├─ 失手摔碎玉佩 → 光球飞出 → 撞入眉心
├─ 功法+药方一次性灌顶
│   ├─ 炼体十八式
│   └─ 回春药方（8味药材）
└─ 第3章《探寻》

永熙十年（1377年）
│
├─ 肖凡7岁
├─ 夜里梳理脑中功法：
│   ├─ 总纲：炼体十八式，三段（每段六式），返老还童每80年一次共3次
│   ├─ 第一段六式：托天、推山、转腰、压腿、开胯、摇臂
│   ├─ 第二三段被封锁，无法读取
│   ├─ 更深内容：经脉、气感、外界的沟通、感知天地（看不懂）
│   ├─ 药方：8味药材（茯苓、赤芍、黄精、人参、当归、何首乌、藏红花……）
│   └─ 附注：此方用于回春，每八十载一次，共历三次方得圆满
├─ 意识到功法与本世界修炼体系完全不同
├─ 决定保密
└─ 第4章《功法》

│
├─ 次日，肖凡找师父询问修炼和长生
├─ 无为子讲解本世界修炼体系：
│   ├─ 横练：铁皮→铜骨→金身（累积暗伤，单修活不久，金身时普通兵刃难伤）
│   ├─ 内力：气感→周天→化罡（滋养暗伤、延年益寿，化罡可外放真气，但身体承载有极限）
│   ├─ 双修互补：横练承载更浑厚内力，内力感知助力精准打熬筋骨，两道皆至顶方有机会感应宗师机缘
│   └─ 宗师：横练金身+内力化罡+机缘，寿命上限约120岁
├─ 师父明确否定长生：从古到今没有人做到过
├─ 肖凡内心对比：本世界耗自身 vs 脑中功法引外界灵气
├─ 确信脑中功法是另一条路，匹夫怀璧，必须保密
└─ 第5章《境界》

│
├─ 当夜，肖凡偷偷溜到后院第一次练习炼体六式
├─ 动作远比想象中困难，但小孩身体柔韧性是优势
├─ 练完后躺床上萌生念头：武功再高也怕火药
├─ 次日找师父打听爆炸粉末→得知石硫黄（硫磺）药铺有售
├─ 硝石无人认识，肖凡决定从老墙根和旱厕自提
└─ 第6章《初练》

│
├─ 肖凡开始从老墙根和旱厕外墙刮硝土，偷偷在后山山洞里试验土法炼硝
├─ 第一次炼硝失败（火太旺，清液烧干，锅底只剩焦渣）
├─ 第二次压着性子慢熬，锅内析出灰白色结晶，炼硝成功
├─ 肖凡将硝石结晶收进竹筒，藏在山洞深处石板下
├─ 与此同时，感气也取得突破：某夜练完炼体六式，气机下沉腹部，成功引入外界灵气
├─ 感气成功后，肖凡去藏书阁向宋明借药材书（《本草简要》《药性赋》），开始核对药材名
└─ 第7章《初窥门径》

│
├─ 林谋带肖凡第一次下山，前往山脚集镇
├─ 集镇见闻：
│   ├─ 菜贩：顾客嫌菜蔫，老妇不肯降价，争执后顾客离开
│   ├─ 鸡贩：顾客嫌鸡屎脏，老汉不肯让步，最终顾客离开
│   ├─ 鱼贩：顾客嫌鱼不新鲜，鱼贩坚持，最终顾客被噎走
│   └─ 乞丐：胖商人把饼扔地上踩一脚，骂乞丐脏，路人避之不及
├─ 肖凡目睹市井百态，内心受到冲击
├─ 林谋在药铺门口等候，肖凡进济安堂药铺买石硫黄（三文钱）
├─ 回程路上，肖凡暗记集镇布局（铁匠铺、药铺、杂货铺位置）
├─ 回观后问无为子："师父，这个世道究竟是什么样的？"
├─ 无为子答："世道就是这样。无力改变，就只有顺从。"
├─ 肖凡第一次意识到：清虚观的清净，是因为身在山里
└─ 第8章《初见世态》
```
永熙十八年（1385年）春
│
├─ 肖凡15岁，下山历练五个月后抵达通州镇（京城以南约五十里）
├─ 在通州镇客栈住下，要了一碗面一壶茶
├─ 在客栈大堂听说书先生讲京城趣事（醉仙楼、东城官宅、西城王府、北城道观）
├─ 从说书闲话中拼出京城格局：东城多官宅，西城多皇亲，南城最热闹，北城清静
├─ 了解各皇子动向：
│   ├─ 大皇子萧恒：留守京城，代巡六部
│   ├─ 二皇子萧厉：辅政朝堂，与世家往来甚密，江南顾氏为其外戚
│   ├─ 三皇子萧沛：不掺朝政，常驻书院，与文人墨客过从甚密
│   └─ 四皇子萧俊：年幼（6岁），未详述
├─ 夜里回房，检查行李时发现师父无为子给的一封信
├─ 信是写给国师玄尘子（无为子师兄）的引荐信
├─ 师父交信时已当面告知：国师是师兄，可以信任，到京城后把信交给国师
├─ 信的具体内容不向肖凡透露
└─ 第9章《远行》
```
