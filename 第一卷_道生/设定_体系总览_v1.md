## 三、修炼体系（去金手指版·current）

### 3.1 凡俗武道（本世界公开，无灵气）

| 路径 | 三阶 | 说明 |
|------|------|------|
| **横练（外功）** | 铁皮 → 铜骨 → 金身 | 铁皮抗钝器；铜骨抗刀剑；金身寻常兵刃难伤，需内力持续滋养 |
| **内力（内功）** | 气感 → 周天 → 化罡 | 气感感知气机；周天贯通全身经脉；化罡外放成罡气 |

- **双修互补**：内力滋养横练暗伤，横练增强身体承载力以撑起更高内力。
- **单修缺陷**：仅横练→暗伤累积，四五十岁身体崩坏；仅内力→经脉有极限，化罡前止步。
- **宗师**：金身＋化罡，寿命自然上限≈120岁（与常人无异，只是更健康）。全国宗师一只手数得过来。

### 3.2 玉佩功法（肖凡独有·克制写实）

- **来源**：肖凡6岁玉佩碎裂，功法信息灌顶入脑（含炼体十八式、七境界、回春/滋养药方）。玉佩已碎，功法不可复得。
- **本质**：一部**无暗伤双修法**——引外界灵气滋养身体，使横练暗伤彻底化解、内力上限突破凡俗瓶颈。
- **代价机制（金手指的缰绳）**：
  1. 突破需灵气环境/天材地宝，稀缺昂贵；
  2. 每境有"心魔劫"，心性不稳则走火入魔、根基尽毁；
  3. 玉佩已碎，练错一步无补救。
- **外练·炼体十八式**（三段不可越级）：炼血肉(1–6) → 炼筋骨(7–12) → 炼脏腑(13–18)。

### 3.3 内炼·七境界（定义，消除"七境界未定义"矛盾）

| 境界 | 含义 |
|------|------|
| 1 凝灵 | 引灵气入体 |
| 2 化精 | 灵气化精，滋养暗伤 |
| 3 淬体 | 灵气淬炼血肉 |
| 4 易筋 | 重塑筋骨 |
| 5 洗髓 | 净化骨髓 |
| 6 通神 | 神识外放，感知敌意 |
| 7 **归元** | 返璞归真，寿元略增，宗师之上的"半步陆仙"（非永生） |

七境界对应强化横练/内力，使肖凡同阶碾压、可越阶而战。

### 3.4 长生·合理融入版（体系顶点，非永生）

- **达成条件**（三重门槛，缺一不可）：七境界全满(归元) ＋ 炼体十八式全成 ＋ 渡过**三次大劫**（每次需在灵气充沛之地闭关数年，失败即死）。
- **效果**：驻颜不老 ＋ 寿元延至约**200岁**。
- **代价**：①终生受灵气源地束缚，远离则衰退；②每百年需再渡"衰劫"，不过则身死；③驻颜期战力反降（灵气内敛养身）。
- **与前期关系**：肖凡15岁仅入门，长生是"传说级"目标。第一卷只埋伏笔（功法末段附注"此道有成者可驻世久长，然劫数亦重"），绝不干扰爽文节奏。

### 3.5 科技碾压协调（爽文关键）

- 黑火药：对铁皮有效、对铜骨杀伤大减、对金身/化罡宗师基本无效。
- 肖凡优势：**铜骨级防御 ＋ 改良火器**（黑火药提纯、燧发枪），形成错位碾压。
- 后期可衍生"灵气附魔火器"（克制，火器仍是凡物＋灵气附魔，非无敌）。

## 四、功法·炼体十八式（细则，详见设定_炼体十八式.md）

- 三段递进，每段六式，不可跳序、不可越级。
- 第一段·炼血肉（1–6式）：托天、推山、转腰、压腿、开胯、摇臂。
- 第二段·炼筋骨（7–12式）：弓脊、摆尾、盘根、举鼎、摇橹、探海。
- 第三段·炼脏腑（13–18式）：抱元、震腑、捧日、抖身、归元、合道。
- 练习规则：开吸合呼、起吸落落、发力时呼气；每日至少一遍，不可中断超三日。

## 六、势力与反派（暗线压顶织法）

### 6.1 背景暗线（肖凡下山前·1379·永熙十二年）
- 正五品前知州**沈括**卸任归乡，被**黑虎帮**二当家灭门47口 → 朝廷震怒 → 推行**武者登记新政**（武者入籍、流动受控）。
- 此事件在第一卷前期为"江湖传闻/新闻"，是肖凡认知世界的背景；新政在肖凡1385进京后成为经营的**隐性规则**（招护院需武籍）。

### 6.2 明线势力（经商线）
- **钱万山**：京城最大酒楼"醉仙楼"老板，大皇子**东宫**的人。盯上"仙人醉"酒方，派人闹事、威胁。
- **肖凡**：盘下"好家伙"酒坊，酿"仙人醉"，开"天上人间"，与钱万山商战。

### 6.3 暗线势力（政治线，后期爆发）
- **虞国听雨阁**：敌国间谍机构。**苏墨凝第10章被追杀的真凶**（非普通仇家）**，她携涉及听雨阁渗透的密信。
- **三皇子萧沛**：表面不掺朝政、常驻书院，实为隐藏反派；借听雨阁构陷大皇子萧恒、挑二皇子萧厉火并。
- **东宫/钱万山** 与 **听雨阁** 暗中勾结，借商战掩护间谍活动。
- **国师玄尘子**：皇权代表，无为子师兄，肖凡的政治靠山（"有难处可来找我"）。

### 6.4 交织点（暗线压顶）
- 前期：新政是酒坊隐性约束；苏墨凝密信埋听雨阁伏笔；四皇子名单埋政治伏笔。
- 中期：钱万山借新政举报肖凡伙计无籍施压；"天上人间"渐成各方情报交汇舞台。
- 后期（40+章大高潮）：听雨阁发动政变，三皇子借机发难；肖凡以"科技＋修炼"双碾压破局，伏笔北方狄族（未来线）。
